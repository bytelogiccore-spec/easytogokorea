# UI/UX Design: 15_Smart Store Alert (Digital Buzzer)

## 📌 UI/UX 목표 (Goals)
1. **아날로그적 직관성**: 실제 진동벨이나 번호표를 쥐고 있는 듯한 확실하고 시각적인 피드백(진동, 커다란 번호, 깜빡임).
2. **백그라운드 모니터링 체감**: 앱을 내리고 있거나 다른(지도, 번역기) 화면을 보더라도 언제든 내 순서를 알 수 있다는 안도감 제공.
3. **네오 미니멀리즘**: 불필요한 텍스트를 줄이고, 대형 타이포그래피와 상태(대기 중/완료)에 따른 명확한 색상 변화.

## 📱 주요 화면 구성 (Key Screens)

### 1. 스마트 진동벨 팝업 (Smart Buzzer Modal / Bottom Sheet)
- **발진 형태**: 가맹점에서 QR/NFC를 스캔하여 주문/웨이팅을 등록하는 즉시, 앱의 어떤 화면에 있든 하단에서 "**Smart Buzzer**" 모달이 올라오거나, 화면 상단에 떠 있는 플로팅 위젯(Floating Widget) 형태로 자리잡습니다.
- **대형 대기 번호**: 화면 중앙에 큼지막하게 대기 번호(예: `No. 42`)를 표시.
- **상태 애니메이션**:
  - `대기 중 (Waiting)`: 번호 주변으로 부드러운 물결(Ripple) 애니메이션이나 은은한 펄스(Pulse) 효과 (색상: 블루/그레이 톤).
  - `준비 완료 (Ready)`: 화면 전체의 배경색이 강렬하게 변하며(예: 네온 그린/오렌지), 햅틱 진동이 아주 강하게 연속으로 울림. 화면 중앙에는 "Pick Up Now! (수령해주세요!)"라는 텍스트가 깜빡임.
- **주문 내역 요약**: 번호 아래에 작게 주문한 메뉴(예: Iced Americano x2)와 가게 이름(명동 커피) 표시.

### 2. 라이브 액티비티 / 다이나믹 아일랜드 연동 (OS Integration)
- 앱 밖으로 나가더라도, iOS 잠금 화면이나 상단 아일랜드에 "명동 커피 대기열 (내 앞 2팀)" 형태로 작게 떠 있음.
- 차례가 되면 전체 화면 알림과 거대한 팝업으로 주의를 집중시킴.

### 3. 주변 탐색 유도 보드 (Explore While You Wait)
- 진동벨 화면 하단에 `[웨이팅 중에 여기 다녀오는 건 어때요?]` 형태의 가벼운 슬라이드 카드 제공.
- 10분 이상 대기가 예상되면 반경 500m 이내의 소품샵, 포토존 등을 추천.

## 👆 터치 인터랙션 (Interactions)
1. **스와이프 업 (수령 완료 처리)**: "준비 완료" 알림이 울릴 때, 화면을 위로 강하게 스와이프 하거나 `[수령 완료]` 슬라이더를 밀어서 알림(진동)을 명시적으로 종료시킬 수 있음 (실제 진동벨을 알바생에게 반납하는 행위를 아날로그적으로 오마주).
2. **위젯 최소화**: 진동벨 팝업을 아래로 스와이프하면, 앱 내 플로팅 아이콘(알약 모양)으로 축소되어 오른쪽 상단에 떠다님. 언제든 터치하면 다시 크게 볼 수 있음.

## 🎨 시각적 특징 (Visual Identity)
- **빛과 진동의 시각화**: 어두운 테마(Dark Mode) 지원 시 대기 번호가 네온사인처럼 밝게 빛나는 효과.
- **색상 기반 상태 인지**: 
  - 기본 대기: `#E5E5EA` (Neutral)
  - 차례 다가옴(내 앞 1팀): `#FFCC00` (Warning/Attention)
  - 준비 완료: `#FF3B30` 또는 `#4CAF50` (Action Required) 맹렬한 점멸 애니메이션.
