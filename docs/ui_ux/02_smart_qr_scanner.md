## 📷 QR 스캔 화면 (Camera View)

하단 탭 중앙의 커다란 `[Scan]` 버튼을 누르면 즉시 카메라 화면이 전체로 열립니다. 
앱은 어떤 QR(전봇대, 식당 메뉴판, 특산품)을 스캔했는지 **자동으로 판별하여 다음 동작을 결정**합니다.

```text
╭───────────────────────────────────────╮
│  ( X )                         ( ❓ ) │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │                                 │  │
│  │             ╭───────╮           │  │
│  │             │   QR  │           │  │
│  │             │ 스 캔 │           │  │
│  │             ╰───────╯           │  │
│  │                                 │  │
│  │  "가로등 번호판이나 식당 메뉴의 │  │
│  │   QR을 사각형 안에 맞춰주세요"  │  │
│  │                                 │  │
│  │        ( ⌨️ 번호 직접 입력 )    │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ ( 🔦 플래시 )     ( 🧩 내 QR )  │  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```

## 🚨 안심 위치 확인 (QR Scan: 가로등/전봇대 인식 시)

길을 잃었을 때 가로등이나 건물 벽의 QR코드를 스캔하면 나타나는 화면입니다.

```text
╭───────────────────────────────────────╮
│  📍 현재 내 위치 확인 완료!           │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ [ 🗺️ 미니 맵이 부드럽게 표시됨 ]│  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  주소: 서울특별시 종로구 세종대로 175 │
│  (GPS: 37.5704° N, 126.9760° E)       │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ ( 🚨 112 / 119 긴급 문자 신고 ) │  │
│  │     "도움이 필요하신가요?"      │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ( 🚕 카카오T 호출 ) ( 🔗 공유하기 )  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │   ( 🚶 AR 도보 길안내 시작하기 )│  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```

### 🚨 119 SOS 및 카카오T 호출 팝업 (Action Sheet)

🚨 긴급 문자 버튼이나 🚕 택시 호출 버튼을 눌렀을 때 나타나는 하단 팝업(Action Sheet)입니다.

```text
╭───────────────────────────────────────╮
│  [ 배경 블러 처리 (Dimmed) ]          │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🚨 119 긴급 문자 발송           │  │
│  │                                 │  │
│  │  "Emergency! Location:          │  │
│  │   Seoul Jongno-gu Sejong-daero 175  │
│  │   (37.5704N, 126.9760E)         │  │
│  │   I need medical help."         │  │
│  │                                 │  │
│  │       [ ✉️ 문자 전송하기 ]      │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🚕 택시 출발지로 지정           │  │
│  │                                 │  │
│  │ [현재 위치]를 출발지로 설정하여 │  │
│  │ 카카오T 앱을 실행합니다.        │  │
│  │                                 │  │
│  │       [ 🚕 카카오T 열기 ]       │  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```

## 🍽️ 식당 메뉴 번역 및 결제 (QR Scan: 테이블 메뉴판 인식 시)

식당 테이블에 붙어 있는 전용 QR을 스캔했을 때 나타나는 주문 화면입니다.

```text
╭───────────────────────────────────────╮
│  🏪 할머니 뼈해장국 (돼지고기)        │
│                                       │
│  메뉴                   ( English ⌵ ) │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │   🍲 Pork Bone Stew (뼈해장국)  │  │
│  │      Spicy soup with tender...  │  │
│  │      ₩ 10,000      ( -  1  + )  │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │   🍚 Extra Rice (공기밥)        │  │
│  │      ₩ 1,000       ( -  0  + )  │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ Total: ₩ 10,000                 │  │
│  │ ( 💳 장바구니 담고 결제 (Pay) ) │  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```
