## 🗣️ 실시간 통역 및 헬프데스크 (Interpreter Matching)

응급 상황, 약국, 관공서 등에서 의사소통이 어려울 때 실시간 전화를 통해 통역사와 연결되는 화면입니다.

```text
╭───────────────────────────────────────╮
│  <  도움 요청 (Help & Interpretation)  │
│                                       │
│  무엇을 도와드릴까요?                 │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🗣️ 실시간 전화 통역 연결        │  │
│  │  - 기본 5분 무료 (이후 분당 500원) │  │
│  │  - 지원: 영어, 중국어, 일본어...   │  │
│  │                                 │  │
│  │       [ 📞 통역사 즉시 호출 ]   │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🚨 긴급 연락처 (원터치 통화)    │  │
│  │  [ 🚓 112 경찰 ] [ 🚒 119 소방 ]│  │
│  │  [ ℹ️ 1330 관광통역안내 (무료) ]│  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```

## 📋 장기 체류 및 비자 가이드 (Long-term Stay Guide)

유학생, 취업자, 워킹홀리데이 등 한국에 장기 체류하는 외국인을 위한 생활 필수 정보 가이드입니다.

```text
╭───────────────────────────────────────╮
│  <  장기 체류 가이드 (Living in KR)    │
│                                       │
│  [ 비자 & 신분증 ] [ 금융 ] [ 주거 ]  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🪪 외국인등록증 (ARC) 발급 안내 │  │
│  │  - 발급 절차 및 필요 서류       │  │
│  │  - 출입국사무소 방문 예약 연동  │  │
│  │             [ 하이코리아 예약 🔗]│  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🏦 은행 계좌 개설 가이드        │  │
│  │  - 외국인 개설 가능 은행 목록   │  │
│  │  - 필수 지참 서류 리스트 체크   │  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```

### 사용자 경험 및 설계 의도 (UX Intent)
*   **직관적인 무료/유료 안내**: 통역 호출 시 기본 무료 시간과 이후 발생하는 비용을 명확하게 팝업으로 고지하여 사용자의 결제 불안을 해소합니다.
*   **장소 기반 맥락 버튼**: 사용자의 현재 GPS가 '약국'이나 '병원' 근처일 경우 '의료 전문 통역' 버튼이 우선적으로 상단에 배치되도록 컨텍스트 어웨어(Context-aware) UI를 적용합니다.
