## 📱 메인 홈 화면 (Main Dashboard)

앱 실행 시 가장 먼저 표시되는 화면입니다. "입력 최소화" 원칙에 따라, 여행 전/중 상황에 맞춘 카드가 자동 추천됩니다.

```text

      ( � 궁궐 탐험가 )          ( 🔍 )
      
      
  Hello, Alex! 👋
  
  
      ( ⛅ 22°C )         ( 💳 ₩ 45,000 )
      
      
  오늘의 추천 💡

    ( 찜질방 양머리 수건 만들기 )
    ( 한국의 귀여운 찜질방 문화를... )
    
    
  빠른 실행

    ( 🏯 템플스테이 )   ( 🎫 K-POP 공연 ) 
    ( 🛒 특산품 직구 )   ( 🚌 축제 셔틀   )
    
                                       
                                       
          ( 🗺️ )    ( QR )    ( 👤 )     
           Home      Scan    Profile   

```

### 화면 구성 요소 설명

1. **상단 프로필바**: 
   - `[12_smart_qr_location.md]`의 성지 순례 스탬프 랠리를 통해 획득한 '업적 배지(👑 궁궐 탐험가 등)'가 이름 옆에 표시됩니다.
2. **콘텍스트 카드 (Context Card)**:
   - 기상청 공공 API 연동 날씨 요약 및 선불/충전된 원화 지갑 잔액 표시.
3. **문화 팁 스와이프 피드**: 
   - `[03_travel_tips.md]` 연계. 카드뉴스 형태로 한국 여행 꿀팁(식당 예절, 찜질방, 대중교통 등)이 노출되며, 좌우로 스와이프하며 넘겨볼 수 있습니다. 틱톡/릴스처럼 짧은 영상 클립도 자동 재생 가능.
4. **빠른 실행 아이콘 (Quick Actions)**:
   - 수익 창출의 핵심인 투어/체험 카테고리로 1터치 진입.
5. **중앙 플로팅 [QR Scan]**:
   - 길을 잃었을 때나 식당 메뉴판 앞에서 즉시 카메라를 켤 수 있도록 가장 크고 눈에 띄게 배치.
