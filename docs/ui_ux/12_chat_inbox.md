## 💬 메시지 인박스 (Message Inbox)

여행사, 통역사, 그리고 플랫폼 AI와의 모든 대화 이력을 모아보는 통합 인박스 화면입니다.

```text
╭───────────────────────────────────────╮
│  메시지 (Messages)               [🔍] │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🤖 EasyToGo AI 비서 (온디바이스)│  │
│  │  "내일 홍대 일정 추가해드렸어.."│  │
│  │                     [ 방금 전 ] │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🏢 하나투어 K-POP 패키지 담당자 │  │
│  │  "네, 셔틀버스 탑승 위치는..."  │  │
│  │                     [ 오전 10:30]│  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🗣️ 김민수 통역사 (매칭 완료)    │  │
│  │  "약국 도착하시면 통화 버튼..." │  │
│  │                     [ 어제 ]    │  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```

## 🤖 AI 통합 검색 및 챗봇 (AI Assistant)

인박스 최상단의 AI 비서 또는 홈 화면의 검색창에서 텍스트나 음성으로 자연어 명령을 내리는 뷰입니다. 기기 내장 AI(Gemini Nano, Apple Intelligence)가 맥락을 분석해 즉시 행동을 수행(라우팅)합니다.

```text
╭───────────────────────────────────────╮
│  <  🤖 EasyToGo AI 비서                │
│                                       │
│              [ 나 ]                   │
│      "비가 오는데 실내 문화 체험      │
│       할 만한 곳 추천해줘"            │
│                                       │
│  [ AI 비서 ]                          │
│  "비 오는 날에는 이런 곳을 추천해요!  │
│   바로 예약 화면으로 이동할까요?"     │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🏛️ 국립중앙박물관 K-컬처 투어   │  │
│  │ ⛩️ 조계사 실내 템플스테이       │  │
│  │       [ 🎟️ 예약 화면으로 이동 ] │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  [ 🎙️ (음성 입력) | ⌨️ 메시지 입력... ]│
╰───────────────────────────────────────╯
```

### 사용자 경험 및 설계 의도 (UX Intent)
*   **자연어 기반 라우팅 (Intent Routing)**: 사용자가 복잡한 메뉴를 찾아다닐 필요 없이 다국어로 "배가 아파", "환전소 어딨어?"라고 입력하면 즉시 관련 화면이나 행동으로 건너뜁니다.
*   **옴니채널 고객센터**: 파트너(여행사)와의 1:1 대화 내역이 앱 내뿐만 아니라 사용자가 연동해둔 외부 메신저(WhatsApp 등)와 실시간으로 동기화됩니다.
