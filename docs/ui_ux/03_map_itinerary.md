## 🗺️ 인터랙티브 지도 화면 (Map View)

사용자의 현재 일정과 주변 관광지, 그리고 특별 레이어(템플스테이, K-POP 성지 등)를 한눈에 볼 수 있는 지도 화면입니다. 백그라운드 데이터는 오프라인 기반으로 동작합니다.

```text
╭───────────────────────────────────────╮
│  ( Profile )      ( 🔍 장소/경로 검색)│
│                                       │
│  ( 🎫 K-POP 성지 )  ( 🏯 전통 체험 )  │
│                                       │
│             ( 💳 NFC 패스 결제 활성 ) │
│                                       │
│            ( 🏯 조계사 )              │
│                  📍                   │
│                   \                   │
│                    \ 15 min           │
│                     \                 │
│     ( 🥩 비건 식당 ) \                │
│           📍----------🔵 (현재위치)   │
│                                       │
│           ( 🧭 내 위치로 이동 )       │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 👇 내 일정 (Day 2: 서울 도심 )  │  │
│  │                                 │  │
│  │ 10:00 AM 🔵 출발 (안국역)       │  │
│  │ 10:15 AM 📍 조계사 (템플스테이) │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │   ( 🗺️ )     ( QR )     ( 👤 )   │  │
│  │    Map       Scan     Profile   │  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```

## 📅 일정 플래너 편집 (Planner Drag & Drop)

위 화면의 하단 패널 `[👇 내 일정]`을 위로 스와이프업(Swipe-up) 했을 때 보이는 전체 화면입니다. Valhalla 라우팅 엔진이 최적의 동선을 자동 계산해 줍니다.

```text
╭───────────────────────────────────────╮
│  ( < 뒤로 )   내 여행 폴더   ( 저장 ) │
│           Day 2: 서울 도심 투어       │
│                                       │
│  ( 🔄 자동 최적화 )  ( ➕ 장소 추가)  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ = 📍 조계사 (09:00 - 11:30)     │  │
│  │      도보 15분 / 대중교통 10분  │  │
│  ╰─────────────────────────────────╯  │
│      │                                │
│  ╭─────────────────────────────────╮  │
│  │ = 🥩 발우공양 비건 식당         │  │
│  │      12:00 예약 확정            │  │
│  ╰─────────────────────────────────╯  │
│      │                                │
│  ╭─────────────────────────────────╮  │
│  │ = 🏯 경복궁                     │  │
│  │      오후 휴무 - ⚠️ 일정 경고!  │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ ( 📥 친구와 일정 공유 (초대) )  │  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```
