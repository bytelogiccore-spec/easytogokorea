# 11. 유저 프로필 및 디지털 월렛 (Profile & Wallet)

> **원칙**: *"나의 여행 기록, 수집한 뱃지, 그리고 모든 결제와 티켓이 안전하게 보관되는 나만의 여권 앱"*

---

## 👤 프로필 대시보드 및 업적 (Profile Dashboard)

사용자의 레벨과 여행하면서 성취한 뱃지, 커뮤니티 활동을 한눈에 볼 수 있는 공간입니다.

```text
╭───────────────────────────────────────╮
│  [설정 ⚙️]                    [QR 💳] │
│                                       │
│       ╭──────────────╮                │
│       │   (프사)     │  Level 4      │
│       ╰──────────────╯                │
│       Alex_Wanderlust                 │
│       "Traveler from UK 🇬🇧"           │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🏆 나의 획득 뱃지               │  │
│  │  [👑궁궐 탐험가]  [🍜매운맛 마스터]│  │
│  │  [🎭인디 서포터]  [+ 빈 슬롯 4]    │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  [ 내 후기 (12) ]  [ 찜한 장소 (34) ]   │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 📋 오프라인 저장 데이터 관리      │  │
│  │  서울 지도 다운로드 완료 (140MB)    │  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```

### 1. 사용자 경험 및 설계 의도 (UX Intent)
*   **게이미피케이션 (Gamification)**: 성지 방문, 식당 스캔, K-POP 후원 등 앱 내 상호작용을 할 때마다 `[배지]`를 부여하여 여행의 성취감을 제공하고, 이 배지가 소셜 피드에 노출되어 커뮤니티 참여를 유도합니다.
*   **오프라인 모드 접근 (Offline First)**: 데이터 로밍이 끊기거나 터널을 지날 때를 대비해, 텍스트 형태의 팁과 지도를 수동으로 관리하고 다운로드 받을 수 있는 캐시 뷰를 제공합니다.

---

## 💳 디지털 패스 월렛 (Smart Wallet)

예약한 템플스테이 티켓, K-POP 라이브 QR, 구매한 교통 패스를 모아두는 곳입니다.

```text
╭───────────────────────────────────────╮
│  <  내 지갑 (Passes & Tickets)         │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🇰🇷 Korea Tourism Pass (3 Days)  │  │
│  │  잔여: 42시간 15분              │  │
│  │  [ NFC 로 대중교통 태그하기 ]   │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  예약된 체험 / 티켓:                   │
│  ╭─────────────────────────────────╮  │
│  │ 🎫 홍대고양이 인디 라이브       │  │
│  │    3/15 19:00                   │  │
│  │          [ QR로 입장하기 ]      │  │
│  ╰─────────────────────────────────╯  │
│  ╭─────────────────────────────────╮  │
│  │ 🏯 조계사 템플스테이 (1박)      │  │
│  │    3/20 14:00                   │  │
│  │          [ 예약 내역 보기 ]     │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│       [ + 새 결제 수단 추가하기 ]     │
╰───────────────────────────────────────╯
```

### 2. 사용자 경험 및 설계 의도 (UX Intent)
*   **카드 스택 디자인 (Card Stack UI)**: iPhone의 Apple Wallet이나 삼성페이처럼 실제 카드가 겹쳐 있는 것 같은 세로 스택형 UI를 사용하여 시각적 친숙함을 줍니다.
*   **NFC 선탑재 (NFC First)**: 교통카드(티머니 등) 기능이 연동된 패스의 경우 앱을 켜지 않고 폰만 흔들거나 가까이 대어도 작동하도록 백그라운드 연동을 강조합니다.

## ⚙️ 설정 및 외부 연동 (Settings & Integrations)

프로필 화면에서 `[설정 ⚙️]` 버튼을 누르면 진입하는 상세 설정 화면입니다.

```text
╭───────────────────────────────────────╮
│  <  설정 (Settings)                     │
│                                       │
│  [ 계정 관리 ]                        │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 💬 외부 메신저 연동 (CS/알림용) │  │
│  │  [ ✔️ WhatsApp 연결됨 ]         │  │
│  │  [ + LINE 계정 연결하기 ]       │  │
│  │  [ + WeChat 계정 연결하기 ]     │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🧠 오프라인 AI 다국어 팩 관리   │  │
│  │  [ 🇰🇷 한국어 (기본) - 설치됨 ]  │  │
│  │  [ 🇺🇸 영어 - 설치됨 (30MB) ]    │  │
│  │  [ 🇯🇵 일본어 - 대기중 ⬇️ ]      │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  [ 알림(Push) 설정 ]                  │
│  [ 테마 및 디스플레이 ]                 │
╰───────────────────────────────────────╯
```

### 3. 사용자 경험 및 설계 의도 (UX Intent)
*   **글로벌 메신저 통합 (Omnichannel)**: 여행자가 익숙한 WhatsApp이나 LINE을 연결해두면, 앱에 접속하지 않아도 예약 알림이나 여행사와의 상담을 본인의 메신저로 계속 이어갈 수 있습니다.
*   **온디바이스 AI 언어팩 (On-device Translation)**: 오프라인 환경에서도 메뉴판 번역이나 AR 길안내 텍스트가 실시간으로 번역되도록 시스템 레벨의 언어팩 다운로드를 관리합니다.
