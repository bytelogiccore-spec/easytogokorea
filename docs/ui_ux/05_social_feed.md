## 📸 피드 탐색 메인 화면 (Feed View)

인스타그램과 유사한 형태로, 여행객들이 자신의 경험을 공유하고 서로 상호작용하는 공간입니다.

```text
╭───────────────────────────────────────╮
│  ( Profile )  ( 서울 피드 ⌵ )  ( 🔍 ) │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ ( 📝 내 여행후기 쓰기 )         │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ @Sarah_NY (👑)      ( ➕ Follow ) │  │
│  │  📍 광장시장 (Gwangjang Market) │  │
│  │                                 │  │
│  │ [         Image Area        ]   │  │
│  │ [        (1/3 photos)       ]   │  │
│  │                                 │  │
│  │ ( ❤️ ) ( 💬 ) ( 📨 )      ( 🔖 )│  │
│  │ 좋아요 1.2k개                   │  │
│  │                                 │  │
│  │ @Sarah_NY: Finally tried the    │  │
│  │ famous mung bean pancake!...    │  │
│  │ ( 🌐 한국어로 번역 보기 )       │  │
│  │                                 │  │
│  │  #FoodieSeoul #GwangjangMarket  │  │
│  │                                 │  │
│  │ (👉 내 일정에 '광장시장' 추가하기)│  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │   ( 🗺️ )     ( QR )     ( 👤 )   │  │
│  │    Home       Scan     Profile  │  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```

## 🏆 내 프로필 및 업적 배지 (Profile & Badges)

스탬프 랠리나 다양한 활동을 통해 획득한 칭호(배지)를 보여주고 장착하는 화면입니다.

```text
╭───────────────────────────────────────╮
│  ( < 뒤로 )    내 프로필       ( 편집 )│
│                                       │
│  ╭─────────────────────────────────╮  │
│  │        ( Profile Image )        │  │
│  │      @Alex_From_Miami           │  │
│  │                                 │  │
│  │    현재 대표 칭호: ( 👑 궁궐 )  │  │
│  │                                 │  │
│  │ ( 게시물 12 ) ( 팔로워 340 )    │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  🎖️ 나의 업적 배지 컬렉션 (Badges)    │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ ( 👑 궁궐 탐험가 ) 장착중       │  │
│  │   - 서울 시내 5대 궁궐 QR 완료  │  │
│  ╰─────────────────────────────────╯  │
│  ╭─────────────────────────────────╮  │
│  │ ( 🎧 BTS 서울 마스터 )          │  │
│  │   - 뮤비 촬영지 3곳 방문        │  │
│  ╰─────────────────────────────────╯  │
│  ╭─────────────────────────────────╮  │
│  │ ( 🍲 길거리 음식 마스터 )       │  │
│  │   - 전통시장 퀘스트 5회 성공    │  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```
