# 06. 축제 정보 및 편도/왕복 셔틀 버스

> **원칙**: *"복잡한 축제 스케줄과 낯선 지방의 대중교통 문제를 '묶음(Bundle) 패키지' UI로 가장 직관적으로 해결한다."*

---

## 🚌 셔틀 버스 예약 및 축제 상세 화면 (Shuttle & Festival)

지방에서 열리는 낯선 축제(보령 머드 축제, 진해 군항제 등)를 완벽하게 즐기기 위한 전용 정보 및 예약 카드입니다.

```text
╭───────────────────────────────────────╮
│  <  진해 군항제 (벚꽃 축제)        🔗  │
│  ╭─────────────────────────────────╮  │
│  │                                 │  │
│  │    [ 화려한 벚꽃 축제 영상 ]    │  │
│  │                                 │  │
│  ╰─────────────────────────────────╯  │
│  📍 경상남도 창원시 진해구             │
│  📅 2026.03.25 - 04.05             │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🚌 왕복 셔틀 버스 묶음 패키지   │  │
│  │    서울 (명동/홍대/강남) 출발     │  │
│  │  출발: 08:00 AM | 복귀: 19:00 PM│  │
```

### 1. 사용자 경험 및 설계 의도 (UX Intent)
*   **시각적 몰입감 (Visual Hook)**: 복잡한 텍스트보다 커버 영역에서 축제 고화질 영상 또는 사진을 재생시켜 여행 뽐뿌를 자극합니다.
*   **패키지 유도 (Bundling)**: "셔틀 버스 예약" 카드를 메인 CTA 컴포넌트로 가장 눈에 띄게 배치하여 교통 불편에 대한 외국인의 두려움을 해소합니다.
*   **상황 인지 데이터 (Contextual Data)**: 기상청 API와 연동된 현지 날씨를 바탕으로 외국인(특히 열대 기후 국가)에게 맞춤형 복장 팁(가디건 필수)을 직관적으로 제안합니다.

## 🎫 디지털 탑승권 월렛 화면 (Shuttle Boarding Pass)

결제 및 예약이 완료되면 생성되는 모듈형 티켓으로, QR코드와 실시간 버스 위치를 제공합니다.

```text
╭───────────────────────────────────────╮
│  <  내 탑승권 (Boarding Pass)          │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🌸 진해 군항제 왕복 셔틀        │  │
│  │ 👨‍👩‍👦 성인 2명                  │  │
│  │                                 │  │
│  │   [ 출발 ]          [ 도착 ]    │  │
│  │    08:00     🚌     12:30       │  │
│  │    홍대입구역         진해역    │  │
│  │                                 │  │
│  │   ┌─────────────────────┐       │  │
│  │   │      ( QR CODE )    │       │  │
│  │   └─────────────────────┘       │  │
│  │     탑승 시 기사님께 스캔       │  │
│  │                                 │  │
│  │  📍 [실시간 버스 위치 확인]     │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  [ 취소/환불 규정 ]                    │
╰───────────────────────────────────────╯
```

### 2. 사용자 경험 및 설계 의도 (UX Intent)
*   **단일 정보 집중 (Single Focus Focus)**: 복잡한 탑승구, 좌석 번호 등을 시각적으로 그룹화하고 가장 핵심적인 **출발 시간/장소와 탑승용 QR코드**를 가운데에 극단적으로 배치합니다.
*   **실시간 안심 (Real-time Relief)**: 지도 연동을 통해 외국인들이 약속된 승차 장소에 버스가 어디쯤 오고 있는지 `[실시간 버스 위치 확인]` 버튼으로 볼 수 있도록 하여 심리적 안정감을 제공합니다.
