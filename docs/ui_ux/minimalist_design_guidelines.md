# EasyToGo Minimalist UI Design Guidelines 🎨

> **목적**: 외국인 관광객이 언어 장벽 없이 직관적으로 사용할 수 있는, 2026년 트렌드에 맞는 미니멀리즘 UI/UX 원칙 정의

## 1. 핵심 원칙 (Core Principles)

### 1) 여백의 극대화 (Ample White Space)
*   **목적**: 인지 부하 감소 및 시선 유도
*   **적용**: 화면에 많은 정보를 채워 넣으려는 시도를 버리고, 콘텐츠 주변에 충분한 여백(Negative Space)을 배치하여 숨 쉴 공간을 제공합니다.
*   **외국인 타겟 효과**: 낯선 환경(한국)에서 복잡한 화면은 피로도를 높입니다. 여백은 사용자에게 심리적 안정감을 줍니다.

### 2) 제한된 색상 팔레트 (Limited Color Palette)
*   **목적**: 브랜드 정체성 구축 및 시각적 노이즈 제거
*   **적용**: 무채색(흑/백/회색)을 기본 배경 및 텍스트로 사용하고, EasyToGo의 **주조색(Primary Color) 하나만 포인트로 사용**합니다 (예: 중요한 버튼, AR 강조 표시).
*   **규칙**: 한 화면에 3가지 이상의 색상을 쓰지 않습니다 (사진/이미지 제외). 다크 모드를 기본 지원합니다.

### 3) 타이포그래피로 계층 구조 생성 (Typography-Driven Hierarchy)
*   **목적**: 선이나 박스 없이 정보의 중요도 안내
*   **적용**: 가독성이 뛰어난 산세리프 폰트(예: Inter, Roboto)를 사용합니다. 폰트의 크기와 두께(Weight)만으로 제목, 본문, 부가 설명을 명확히 구분합니다.
*   **다국어 고려**: 한국어, 영어, 일본어 등 언어가 바뀌어도 레이아웃이 깨지지 않도록 가변 폰트(Variable fonts) 및 넉넉한 텍스트 박스를 설계합니다.

### 4) 제로 클릭 & 직관적 네비게이션 (Zero-Click & Intuitive Navigation)
*   **목적**: 사용자가 고민할 시간을 없앰
*   **적용**: 햄버거 메뉴를 최소화하고, 화면 하단이나 스와이프 제스처만으로 앱의 핵심 기능(AR 스캐너 등)에 접근하게 합니다.
*   **스마트 디폴트**: 위치 정보를 기반으로 사용자가 지금 가장 필요로 할 기능(예: 지하철역 근처면 교통카드 화면 팝업)을 먼저 띄워줍니다.

## 2. UI 컴포넌트 디자인 (Component Design)

### 1) 버튼 및 인터랙션
*   **스타일**: 그림자 효과나 화려한 그라데이션을 뺀 플랫(Flat) 하거나 미세한 깊이감(Soft Depth)만 있는 버튼.
*   **마이크로 인터랙션**: 버튼을 누를 때 햅틱 피드백과 함께 부드럽게 색이 변하거나 미세하게 크기가 줄어드는(Scale down) 등의 절제된 애니메이션을 사용해 '작동 중'임을 확실히 알립니다.

### 2) 아이콘 및 그래픽
*   **스타일**: 속이 텅 빈 선형(Line) 아이콘을 기본으로 하되, 선택된 상태에서는 면이 채워진(Solid) 아이콘으로 변경하여 상태를 명확히 합니다.
*   **라벨**: 아이콘만으로 의미 전달이 어려울 수 있으니 (특히 다문화 타겟), 아이콘 아래에 간결한 텍스트 라벨을 항상 병기합니다.

### 3) 모달 및 팝업 지양
*   **적용**: 화면을 가리는 팝업이나 모달(Alert) 창은 최대한 지양합니다. 대신 화면 하단에서 부드럽게 올라오는 바텀 시트(Bottom Sheet)나 인라인 알림창(Snackbar)을 활용하여 작업의 연속성을 깨지 않습니다.

## 3. 적용 예시 (Use Cases)

| 화면 | 기존 방식 | 미니멀리즘 적용 |
| :--- | :--- | :--- |
| **홈 화면** | 배너 수십 개, 빽빽한 메뉴 아이콘 | **정중앙에 검색창/AR 스캐너 버튼 1개**, 하단 탭바 메뉴 4개 끝 |
| **결제 화면** | 복잡한 폼, 약관 텍스트 전체 노출 | **총 결제 금액만 대문짝만하게 표시**, 결제수단 아이콘 3개 나열 (원클릭) |
| **식당 메뉴** | 사진+설명+가격+리뷰 혼재 | **여백 상단에 고품질 사진 1장 깔끔하게 배치**, 간결한 제목과 가격만 대비되게 표시 |

## 4. 체크리스트
- [ ] 화면을 딱 봤을 때 가장 중요한 버튼(Call to Action) 1개가 눈에 띄는가?
- [ ] 텍스트를 읽지 않아도 아이콘이나 배치만으로 무엇을 해야 할지 알 수 있는가?
- [ ] 장식 목적의 불필요한 선, 박스, 그림자가 없는가?
- [ ] 다크 모드 전환 시에도 가독성과 심미성이 유지되는가?
