# 08. 특산품 마켓 다국어 언박싱 및 쇼핑

> **원칙**: *"기념품 쇼핑 시 바가지 피해를 막고, 직배송으로 두 손 가볍게 여행을 즐긴다."*

## 🛍️ 스마트 로컬 특산품 마켓 (Local E-commerce)

지역 전통 시장(예: 동대문, 명동 상가)의 매대 상품에도 고유 QR 정보가 부여되며, 이를 스캔하여 번역된 상세 성분과 리뷰를 확인하고 장바구니에 담을 수 있습니다.

│  ╰─────────────────────────────────╯  │
│                                       │
│  [🎁 기념품] [🍜 식품] [💄 브랜드 화장품]│

### 1. 사용자 경험 및 설계 의도 (UX Intent)
*   **다국어 가격 투명성 (Price Transparency)**: 바가지 요금에 대한 우려를 없애기 위해 현지 화폐(KRW)와 앱에 설정된 홈 국가 화폐(USD, EUR 등)를 동시 표시.
*   **안전 인증 뱃지 (Trust Signals)**: 상품 상단에 별점 리뷰와 제조사(브랜드)를 뚜렷하게 배치하여 외국인의 심리적 구매 장벽을 낮춤.

## 📦 출국지/호텔 직배송 예약 화면 (Logistics/Delivery)

물건을 양손 무겁게 들고 다니지 않도록, 구매한 상품을 인천공항 내 픽업 데스크 또는 머무는 호텔 앞까지 당일/익일 배송하는 통합 선택 UI.

```text
╭───────────────────────────────────────╮
│  <  결제 및 배송 방법 (Shipping)       │
│                                       │
│  총 상품 금액: ₩ 55,000               │
│                                       │
│  배송 수령 방법 (Pick-up or Ship):     │
│  ╭─────────────────────────────────╮  │
│  │ 🏨 현재 묵고 있는 호텔로 받기     │  │
│  │   (Shilla Stay Gwanghwamun)     │  │
│  │   오늘 오후 8시 도착 예정       │  │
│  ╰─────────────────────────────────╯  │
│  ╭─────────────────────────────────╮  │
│  │ ✈️ 출국 공항에서 수령하기       │  │
│  │   (인천국제공항 T1 / 수화물 앞) │  │

│  ╭─────────────────────────────────╮  │

│  │   (Fedex / K-Post 연동)         │  │
│  │   배송비 별도 청구              │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│               [ 총 ₩ 55,000 결제 ]    │
╰───────────────────────────────────────╯
```

### 2. 사용자 경험 및 설계 의도 (UX Intent)
*   **Hands-free Travel 원칙**: 여러 개별 상점에서 구매한 물건의 배송 정보를 통합하여, 가장 편리한 수령 방법을 한 번의 스크롤 없는 카드로 제안합니다. 앱의 지향점인 "여행에만 오롯이 집중하는" 환경을 마련합니다.
