## 🎤 K-POP 신인/인디 공연 예매 (Pay-What-You-Want)

신인 아티스트의 공연을 무료(또는 초저가)로 예약하고, 현장에서 '팁'과 '굿즈'를 결제하는 구조를 담은 화면입니다.

```text
╭───────────────────────────────────────╮
│  ( < 뒤로 )  ( 공연 상세 정보) ( 공유)│
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ [ Image : 아티스트 밴드 사진 ]  │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│   ▶️ ( 신곡 하이라이트 미리듣기 15s ) │
│                                       │
│  아티스트: 인디 밴드 '홍대고양이'     │
│  일시: 2026.03.15 (토) 19:00          │
│  장소: 마포구 청년문화공간 (유휴공간) │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ ( 티켓팅: 자율 관람료 Free)     │  │
│  │                                 │  │
│  │ 이 공연은 무료입니다. 관람 후   │  │
│  │ 만족하신 만큼 (팁 Tip)으로      │  │
│  │ 아티스트를 응원해주세요!        │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ( 🎫 티켓 1장 무료 발급받기 )        │
│  ( 🎁 굿즈 함께 미리 주문하기 MD )    │
╰───────────────────────────────────────╯
```

### 아티스트 팁(Tip) 결제 팝업 (공연 종료 후 자동 팝업)

사용자가 공연장 QR을 스캔하고 나가는 시점이나 종료 시간에 맞춰 노출되는 팝업입니다.

```text
╭───────────────────────────────────────╮
│  🎉 '홍대고양이' 공연은 어떠셨나요?   │
│                                       │
│  작은 후원이 아티스트의 꿈을 키웁니다.│
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ ( ☕ 커피 한 잔      ₩ 5,000 )  │  │
│  │ ( 🍔 따뜻한 식사    ₩ 15,000 )  │  │
│  │ ( 직접 입력        ₩ _______ )  │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  ( 💖 Apple Pay로 1초 만에 응원하기 ) │
│  ( 나중에 하기 )                      │
╰───────────────────────────────────────╯
```

## 🏯 템플스테이 4터치 원스톱 예약 (Templestay Booking)

가장 복잡한 템플스테이 예약을 극단적으로 단순화한 카드 선택형 인터페이스입니다.

```text
╭───────────────────────────────────────╮
│  ( < 뒤로 )    템플스테이 예약        │
│                                       │
│  1. 테마를 선택하세요 (스와이프)      │
│  ╭─────────────────────────────────╮  │
│  │ <( 🧘 명상 중심: 참선, 108배 )> │  │
│  │   - 해인사 (경상남도 합천군)    │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  2. 날짜와 인원을 확인하세요          │
│  ╭─────────────────────────────────╮  │
│  │ ( 📅 10.25 - 10.26 / 1박 2일 ⌵) │  │
│  │ ( 👥 성인 2명 ⌵ )               │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  3. 총 결제 금액: ₩ 140,000           │
│  ╭─────────────────────────────────╮  │
│  │ ( 등록된 Visa 카드로 결제: 지문)│  │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```
