## 📸 AR 가이드 및 핀(Pin) 오버레이 (AR Camera View)

스마트 QR 스캐너 앱에서 AR 모드를 켰을 때 실제 카메라 화면 위에 정보가 뜨는 뷰입니다.

```text
╭───────────────────────────────────────╮
│  <  주변 가이드 (종료하려면 탭)        │
│                                       │
│  ( 뒤에는 실제 카메라 영상 배경 )      │
│                                       │
│          ╭────────────╮               │
│          │🏯 경복궁   │               │
│          │   250m     │               │
│          ╰─────┬──────╯               │
│               📍                      │
│                                       │
│                         ╭───────────╮ │
│                         │☕ 한옥카페│ │
│                         │   50m     │ │
│                         ╰─────┬─────╯ │
│                              📍       │
│                                       │
│        [ 3D 화살표 애니메이션 ]       │
│       ▲    ▲    ▲    ▲    ▲         │

│  │ ⤴️ 50m 앞 우회전                │  │

╰───────────────────────────────────────╯
```

### 1. 사용자 경험 및 설계 의도 (UX Intent)
*   **직관적인 플로팅 핀 (Floating Information)**: 건물이나 목적지 방향 허공에 둥둥 뜬 형태의 핀(Pin)을 보여주어, 복잡한 골목길이나 간판이 빽빽한 한국의 도심 속에서도 헤매지 않도록 돕습니다.
*   **다이나믹 화살표 경로 (Haptic/Visual Path)**: 바닥을 쏘는 카메라 뷰에 3D 화살표 이펙트를 배치하여, 사용자가 어느 방향으로 몇 발자국을 더 걸어가야 하는지 즉각적으로 인지하게 합니다.
*   **긴급 접근성 (SOS Button)**: 카메라 촬영 기반의 화면이므로, 만약 낯선 거리에서 위협을 느끼거나 길을 완전히 잃었을 때를 대비해 우측 하단에 상시 `[ 🚨 SOS ]` 버튼을 배치합니다. (112 연동 및 위치/카메라 정보 전송)
