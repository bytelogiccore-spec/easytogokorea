# 09. 병원 예약 및 의료 보험 청구 자동화

> **원칙**: *"복잡한 의료 시스템과 보험 청구를 앱 내에서 완벽히 통제하여 언어적, 행정적 두려움을 해소한다."*

---

## 🏥 외국인 친화 병원 예약 및 통역 (Medical Hub)

여행 중 아플 때 가장 가까운, 그리고 '자신의 언어'가 통하는 병원을 즉시 찾고 예약하는 화면입니다.

```text
╭───────────────────────────────────────╮
│  <  의료 지원 및 병원 찾기 (Medical)   │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🚨 지금 몸이 편찮으신가요?      │  │
│  │    [ 119 긴급 통화 연동 ]       │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  [🔍 내 주변 외국어 가능 병원 찾기]     │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🏥 이화여자대학교 의료원        │  │
│  │    ★ 4.8 (내과, 정형외과)       │  │
│  │    🗣️ English, 🇨🇳 中文 지원      │  │
│  │  📍 신촌역 2번 출구 (1.2km)     │  │
│  │   [ 바로 방문 접수 ]  [ 예약 ]  │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  [ 🗣️ 1:1 실시간 의료 통역사 연결 ]     │
│                                       │
│          보유 보험: Travel Basic      │
╰───────────────────────────────────────╯
```

### 1. 사용자 경험 및 설계 의도 (UX Intent)
*   **긴급성 최우선 (Urgency First)**: 패닉 상태일 수 있는 사용자를 위해 최상단에 빨간색 SOS 긴급 버튼을 배치하여 119 다국어 서비스로 바로 연결되게 합니다.
*   **언어 필터 자동화**: 병원 리스트는 사용자의 설정 언어(예: 영어)를 구사할 수 있는 의료진이 있는 곳을 최우선으로 정렬하여 신뢰도를 높입니다.
*   **온라인 즉시 접수 (Drop-in)**: 복잡한 전화 예약 없이, 한국의 '똑닥' 앱처럼 외국인 번호나 여권 정보로 **바로 방문 접수** 처리하여 병원 대기 시간을 최소화합니다.

## 📄 보험 서류 자동 OCR 청구 (Insurance Claim)

진료 후 발급받은 영수증과 처방전을 카메라로 찍으면, AI가 인식하여 고국의 보험사에 보낼 수 있는 영문 청구 폼을 원터치로 생성해줍니다.

```text
╭───────────────────────────────────────╮
│  <  의료비 청구 서류 만들기            │
│                                       │
│  Step 2/3: 영수증 스캔 결과            │
│  ───────────────────────────────────  │
│                                       │
│  [  📄 의료비 영수증 이미지.jpg  ]     │
│                                       │
│  ✅ **AI 인식 정보 (확인해주세요)**    │
│  ╭─────────────────────────────────╮  │
│  │ 병원명 : 이화여자대학교 의료원  │  │
│  │ 진료일 : 2026.04.12             │  │
│  │ 총 금액: ₩ 115,000 (약 $85)     │  │
│  │ 질병코드: J06.9 (급성 상기도감염)│  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  서류 수신처 (Insurance E-mail)        │
│  [ claim@allianz.com             ]    │
│                                       │
│          [ 영문 청구서 PDF 전송 ]     │
╰───────────────────────────────────────╯
```

### 2. 사용자 경험 및 설계 의도 (UX Intent)
*   **서류 스트레스 제로 (Hassle-free Claim)**: 한국 병원 특유의 복잡한 한글 영수증을 OCR과 공공 의료 데이터(HIRA) API를 통해 읽어내고, 진단 코드를 국제 기준(ICD)으로 번역하여 영문 문서로 재구성해 줍니다.
*   **다이렉트 전송 (Direct Send)**: 생성된 PDF 문서를 개인 폰에 저장할 필요 없이, 사전 등록된 보험사 이메일 폼으로 바로 쏘아주는 원스텝 액션을 제공합니다.
