# 07. 식당 메뉴 번역 및 다국어 결제 화면

> **원칙**: *"QR 코드 하나로 메뉴판, 번역, 결제, 팁까지 모든 식당 내 커뮤니케이션 장벽을 허문다."*

---

## 🍽️ 다국어 스마트 메뉴 주문 (Smart Multilingual Menu)

테이블, 벽, 영수증에 부착된 식당의 로컬 QR 코드를 앱 내 중앙 QR 플로팅 스캐너로 읽었을 때 실행되는 매장 전용 UI 화면입니다.

```text
╭───────────────────────────────────────╮
│  <  광장시장 박가네 빈대떡 (Table 5)     │
│                                       │
│  [🔤 English ⌵]  [🌱 비건 끄기]        │
│  ───────────────────────────────────  │
│  🔥 베스트 메뉴                        │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🟢 녹두 빈대떡 (Mung Bean Pk.)  │  │
│  │    바삭하게 부친 전통 고기 전   │  │
│  │   [사진]    ₩ 15,000            │  │
│  │             [ -  1  + ]         │  │
│  ╰─────────────────────────────────╯  │
│  ╭─────────────────────────────────╮  │
│  │ 🔴 마약 김밥 (Mini Kimbap)      │  │
│  │    겨자 소스가 특징인 한입 김밥 │  │
│  │   [사진]    ₩ 6,000             │  │
│  │             [ -  0  + ]         │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│             [ ₩ 15,000 주문하기 ]     │
╰───────────────────────────────────────╯
```

### 1. 사용자 경험 및 설계 의도 (UX Intent)
*   **매끄러운 다국어 매핑 (Seamless i18n)**: 스캔 즉시 사용자의 폰 기본 설정 언어(예: English)로 메뉴 정보를 렌더링합니다. 한국어 원문이 필요할 수 있으므로 병기하거나 쉽게 토글 가능하게 설계합니다.
*   **문화 필터 버튼 (Dietary Tags)**: 외국인 여행객의 수요가 높은 `[🌱 비건]`, `[알레르기 필터]` 버튼 상단 토글 적용. 비건을 누르면 고기가 들어간 육수/전은 자동으로 회색(비활성) 처리됩니다.
*   **실시간 합계 확인 (Sticky Action)**: 하단 플로팅 버튼에 담긴 금액을 즉각 파악 가능. 주문 시 종업원 호출 시스템과 연동됩니다.

## 💳 분할 결제 및 테이블 팁 결제 (Dutch Pay & Tipping)

식사를 마친 후, 앉은 자리에서 외국인 결제 수단으로 나눠 내거나, 종업원에게 팁을 전송하는 흐름입니다.

```text
╭───────────────────────────────────────╮
│  <  결제하기 (Checkout)                │
│                                       │
│  🟢 통 주문 (Total Bill)               │
│      ₩ 21,000                         │
│  ───────────────────────────────────  │
│  [ 🧑‍🤝‍🧑 친구와 나눠 내기 (Split) ]    │
│  1/2 진행 → 당신은 ₩ 10,500 지불      │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │ 🎉 맛있는 식사셨나요?           │  │
│  │   (선택) 직접 팁을 남겨보세요.  │  │
│  │                                 │  │
│  │   [ No Tip ] [ ₩ 1,000 ]        │  │
│  │   [ ₩ 2,000 ] [ 직접 입력 ]     │  │
│  ╰─────────────────────────────────╯  │
│                                       │
│  결제 수단:                            │
│  [ 💳 Apple Pay / Visa ****1234 ]     │
│                                       │
│              [ 결제 확정하기 ]        │
╰───────────────────────────────────────╯
```

### 2. 사용자 경험 및 설계 의도 (UX Intent)
*   **더치페이 자동화 (Dutch Pay UI)**: 외국 여행객들이 흔히 겪는 "각자 신용카드로 계산하기(Split bill)"의 지연과 혼선을 모바일 상에서 n분의 1 분할 승인으로 원활하게 처리해줍니다.
*   **마찰 없는 팁 문화 지원 (Frictionless Tipping)**: 무인화 상점, 푸드 트럭, 로컬 식당 등에서 서비스 만족 시 앱으로 간편 팁 송금. K-POP 후원 아이디어와 맥락을 같이 하는 자발적 가치 지불 방식 적용.
