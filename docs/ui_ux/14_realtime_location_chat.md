## 🎨 화면 구조 (Screen Layout)

실시간 채널 기능은 완전히 별개의 탭으로 존재하기도 하지만, 주로 지도나 홈 화면 어디서든 **"HUD(Head-Up Display) 오버레이 방식"**으로 접근할 수 있도록 설계합니다.

```text
╭───────────────────────────────────────╮
│  <  🌍 글로벌 지역 채널 (명동)  [≡]  │
│ ╭───────────────────────────────────╮ │
│ │ 📢 [System] 명동 채널에 오신      │ │
│ │             것을 환영합니다.      │ │
│ ╰───────────────────────────────────╯ │
│                                       │
│ [JP] Sakura: 명동 환전소 어디가     │
│              가장 저렴한가요?       │
│                                       │
│ [US] John:  나 지금 이 식당인데      │
│     📍 [명동교자 본점] (라우팅)       │
│                                       │
│ [KR] User:  저도 지금 근처인데!      │
│   🤝 [파티 결제 제안] 1/N 정산       │
│                                       │
│ ╭───────────────────────────────────╮ │
│ │ 💬 메시지를 입력하세요...    [➤]  │ │
│ ╰───────────────────────────────────╯ │
│                                       │
│  ╭─────────────────────────────────╮  │
│  │    ( 📍 )   ( 🌐 )   ( 🎉 )      │ │
│  │     지역     언어     파티        │ │
│  ╰─────────────────────────────────╯  │
╰───────────────────────────────────────╯
```

## 👆 주요 터치 인터랙션 (Interactions)

1. **채널 변경 스와이프**: 하단의 탭(지역, 언어, 파티)을 터치하거나 화면을 좌우로 스와이프하여 채팅 채널을 즉시 전환할 수 있습니다.
2. **지도 위 핀 및 파티 모집**: 입력창 옆의 `+` 버튼을 눌러 내 위치나 보고 있는 식당, 혹은 **"파티 모집(LFG)" 구인 카드** 형식의 UI를 스레드에 띄웁니다.
3. **말풍선 터치 (라우팅 & 참여)**: 누군가 보낸 장소 핀을 터치하면 경로(Valhalla)가 모달로 나타납니다. '파티 모집 카드'를 터치하면 참여(Join) 및 조건 확인 팝업이 뜹니다.
4. **프로필 터치 (동행 매칭)**: 대화 상대의 닉네임을 터치하면 가벼운 프로필 팝업이 뜨고, 사용자의 매너 점수(신뢰도) 확인 및 1:1 대화 요청 기능이 제공됩니다.
5. **파티 1/N 정산 칩**: 동행이 성사되어 그룹이 묶이면, 대화창 내에 "파티 결제(1/n Split)" 버튼 말풍선이 생성됩니다. 대표 1인이 결제 시 그룹원들에게 앱 푸시로 정산 금액이 전달되며, 미리 연결된 수단/포인트에서 즉시 차감할 수 있습니다.

## 🌈 UI 컴포넌트 동작 방식

- **HUD 오버레이 투명도**: 지도 뷰에서 채팅창을 띄우면 채팅창 배경이 50% 반투명(Glassmorphism) 처리되어 뒤쪽의 지도를 보면서 대화할 수 있습니다.
- **다국어 자동 번역 표시**: 메시지 옆에 원래 언어(예: [JP])가 작게 표시되고, 텍스트 자체는 사용자의 설정 언어(예: 한국어)로 번역되어 보여집니다. 번역 아이콘 터치 시 원문을 볼 수 있습니다.
- **채널 상단 바**: 상단에 현재 참여 중인 채널의 이름(예: '글로벌 지역 채널(명동)' 또는 '동일 언어 전체 채널(English)')이 지속적으로 표시됩니다.
