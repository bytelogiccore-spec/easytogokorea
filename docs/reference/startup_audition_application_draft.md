## 1. 문제의식 — 왜 이 서비스가 필요한가?

### 🌏 한국은 미래 도시의 인프라를 갖추고 있지만, 외국인은 체감하지 못합니다

한국은 세계 1위 5G 보급률, 전국 QR 인프라(전봇대·가로등), 98% 카드 결제율, K-의료 경쟁력을 갖춘 **첨단 도시 국가**입니다. 그러나 정작 한국을 찾는 **2,500만 외국인 관광객**은 여전히:

| 불편 | 현실 |
|------|------|
| **메뉴판이 한국어** | 식당 앞에서 망설이다 돌아감 → 소상공인 매출 손실 |
| **예약 불가** | 전화 예약은 한국어만 가능, 온라인 예약 없는 가게가 대다수 |
| **정보 파편화** | 교통·맛집·축제·병원 정보가 각각 다른 앱에 흩어져 있음 |
| **의료 접근 장벽** | 한국 의료가 세계적이지만, 외국인은 병원 찾기·보험 청구·통역이 모두 어려움 |
| **디지털 소통 부재** | 현장의 감동을 공유할 디지털 소통 창구가 없음 |

동시에, 이들을 맞이하는 **소상공인 770만 개**도:

| 고민 | 현실 |
|------|------|
| **외국인 손님 받고 싶지만** | 다국어 메뉴판 제작 비용 부담, 번역 업데이트 어려움 |
| **예약 시스템 도입하고 싶지만** | 네이버·배민 수수료 높고, 월 고정비 부담 |
| **자체 서버 운영 기술 없음** | IT 인력 없어 플랫폼에 종속될 수밖에 없음 |

> **핵심**: 한국의 첨단 인프라와 외국인의 실제 경험 사이에 **거대한 갭**이 존재합니다. EasyToGo는 이 갭을 세 가지로 메웁니다:

> 1. **AR 레이어** — 카메라를 들면 간판·메뉴·지하철이 다국어 / AR로 길안내
> 2. **자체 호스팅 SaaS** — 소상공인이 수수료 없이 외국인 예약·주문을 받음 / 결제 간편화 / QR코드 다국어 메뉴판
> 3. **의료관광 자동화** — 병원 찾기→통역→보험 청구까지 앱 하나로 완결

## 2. 아이디어 — AR로 연결되는 미래 도시

> **"한국을 세계 최초의 'AR 스마트 관광 도시'로 — 소상공인에겐 디지털 자립을, 외국인에겐 미래 도시 경험을"**

### 🏙️ 비전: AR 레이어가 깔린 미래 도시

```
현실 도시 (물리 세계)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  건물  |  간판  |  지하철  |  병원  |  식당
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

       ＋  EasyToGoKorea AR 레이어

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  AR     AR      AR       AR     AR
  낙서  다국어   환승     의료    메뉴판
  ·SNS  간판    네비    안내    ·예약
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

→ 카메라를 들면 도시 전체가 다국어로 살아남
→ 한국어를 몰라도 "보이는 대로 따라가면" 어디든 도착
→ 소상공인 매장이 자동으로 글로벌 비즈니스가 됨
```

### 핵심 기능

**🏪 사업자 앱 — "내 가게의 디지털 자립"**

| 기능 | 설명 | 차별점 |
|------|------|--------|
| **자체 호스팅 서버** | 매장 PC/태블릿이 곧 서버. **서버 비용 제로** | Tauri 기반 → 클라우드 비용 없음 |
| **AR 다국어 메뉴판** | 카메라로 비추면 실시간 번역 + 사진·가격 AR 오버레이 | 메뉴판 교체 불필요, 6개국어 |
| **온라인 예약** | QR 하나로 예약·대기·주문 | 자체 호스팅 → 중개 수수료 0% |
| **AR 간판 광고** | 매장 앞에 AR 프로모션 자동 표시 | 외국인에게 자동 노출 |

**🎨 관광객 앱 — "AR로 체험하는 미래 도시"**

| 기능 | 설명 | 차별점 |
|------|------|--------|
| **3D AR 낙서** | 허공에 3D 텍스트·그림·SNS 링크, 추천 많을수록 커짐(최대 5m) | ARCore Geospatial + Cloud Anchor |
| **🚇 지하철 AR 네비** | 역 내 QR/위치 기반 → 환승·출구 AR 화살표 안내 | BLE 비콘 + 실내 측위, 6개국어 |
| **AR 매장 탐색** | 주변 식당 AR 메뉴판·리뷰·예약 버튼이 카메라에 떠오름 | 사업자 자체 호스팅과 실시간 연동 |
| **AR 성지 스탬프** | K-드라마 촬영지·맛집 QR 스캔 → AR 배지 수집 | 게이미피케이션으로 재방문 유도 |

**🏥 의료관광 마켓플레이스 — "K-메디컬의 쇼피파이"**

에이전시 + 병원 + 환자를 연결하는 **B2B2C 플랫폼**:

```
외국인 환자 (앱 사용자 2,500만 풀)
       ↓ 비교·리뷰·예약
  EasyToGo 의료관광 마켓플레이스
  ┌────────────────────────────┐
  │ 에이전시 A (성형) │ B (검진) │
  │ 병원 C (대학병원 직접 등록)  │
  │ → 비교·리뷰·예약·통역·보험  │
  └────────────────────────────┘
       ↓ 환자 연결
     등록 병원·의료기관
```

| 참여자 | EasyToGo가 주는 가치 | 수익 |
|--------|---------------------|------|
| **외국인 환자** | 에이전시 비교, 가격 투명화, AI 통역, 보험 자동화 | 무료 |
| **의료관광 에이전시** | 2,500만 관광객 풀 노출 + 예약 관리 SaaS | 월 구독 + 성사 수수료 |
| **병원** | 외국인 환자 유입 + 다국어 접수 시스템 | 월 구독 |

| 핵심 기능 | 설명 |
|----------|------|
| **에이전시 스토어** | 에이전시가 시술 패키지·가격·후기를 등록 → 환자가 비교·예약 |
| **보험 청구 자동화** | 진료비 영수증 촬영 → AI OCR → 보험 청구 서류 자동 생성(PDF) → 사용자가 **이메일·메신저로 본국 보험사에 직접 전송**. 서류 작성의 번거로움만 제거 |
| **의료 통역** | 온디바이스 AI 실시간 통역 (오프라인 작동) |
| **의료+관광 일정** | 건강검진 + 관광 일정을 통합 설계 |

> 💡 한국 의료관광 시장은 **연 60만 명** 규모이며, 관광공사 2026 핵심 전략입니다. 그러나 에이전시마다 웹사이트가 따로고 가격이 불투명해 환자도 에이전시도 비효율에 시달리고 있습니다. EasyToGo가 **에이전시·병원·환자를 하나로 연결하는 마켓플레이스**가 되면, 시장 전체의 투명성과 접근성이 바뀝니다.

**📡 통합 정보 허브**

|------|------|
| AI 일정 플래너 | 지도 기반 + 교통비 계산 + 오프라인 작동 (온디바이스 AI) |

## 3. 왜 지금인가? — 시장 기회

### 시장 규모

| 지표 | 수치 |
|------|------|
| 외래 관광객 목표 (2026) | **2,500만 명** |
| 한국 소상공인 수 | **770만 개** |
| 의료관광 시장 | **연 60만 명**, 관광공사 핵심 전략 |
| 관광 수입 목표 | **300억 달러** |

### 정부 정책과 완벽한 타이밍

- **관광공사**: AI 데이터 표준화 + 관광 벤처 육성 + **의료·웰니스 특화** → 우리가 지원 대상
- **제로페이**: 외국인 QR 결제 전국 확대 MOU (2025.5) → 파트너십 추진 가능
- **스마트시티**: 정부 스마트시티 사업과 AR 인프라 연계 가능
- **Open-loop Transit (2030)**: 해외 카드 직접 결제 → 장기 로드맵 포함

### 수익 모델

| 수익원 | 단가 | 특징 |
|--------|------|------|
| **사업자 SaaS 구독** | 식당 1.5만 / 병원 10만 / 여행사 5만 원·월 | **핵심 매출** |
| AR 광고 (빌보드·쿠폰) | CPM / CPC | 현실 공간에 자연스럽게 녹아드는 광고 |
| 의료관광 중개 수수료 | 예약금의 5~10% | 건강검진·시술 예약 연계 |
| AR 프리미엄 소재 | 인앱 판매 | 특수 이펙트·한정판 스티커 |
