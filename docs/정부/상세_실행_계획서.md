### 1-2. Tauri V2 + Svelte 앱 MVP 개발 (3~6월)

#### 프로젝트 구조
```
EasyToGoKorea/
├── app/                           ← [NEW] Tauri V2 + Svelte 프로젝트
│   ├── lib/
│   │   ├── core/                  ← Rust FFI 브릿지, 설정, 테마
│   │   ├── features/              ← 기능별 모듈
│   │   │   ├── onboarding/        ← 언어/국가 선택, 소셜 로그인
│   │   │   ├── map/               ← 지도 + 일정 플래너
│   │   │   ├── info/              ← 관광/날씨/환율 정보 허브
│   │   │   ├── medical/           ← 병원 검색, OCR, PDF 생성
│   │   │   ├── ar/                ← AR 그래피티, AR 내비
│   │   │   └── social/            ← 소셜 피드
│   │   ├── shared/                ← 공통 위젯, 유틸리티
│   │   └── main.dart
│   └── pubspec.yaml
├── api-server/                    ← [기존] Rust/Axum GraphQL
├── partner-app/                   ← [NEW] Tauri V2 파트너 앱
└── docs/                          ← [기존] 기획 문서
```
