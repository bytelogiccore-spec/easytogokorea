# 12. 파트너 포탈 (Partner Portal) 🏢

> **모토**: *"사업자 파트너에게 디지털 운영 도구를 — AR 경로 편집부터 CRM까지"*
> **최종 수정**: 2026-03-11

---

## 📋 1. 개요 (Overview)

파트너 포탈은 **Tauri V2 데스크톱 앱** 기반의 사업자용 백오피스 도구입니다.
기존 매장 관리(SaaS)에 더하여, AR 실내 길안내 경로를 직접 편집·관리할 수 있는 기능을 제공합니다.

### 대상 파트너

| 파트너 유형 | 핵심 기능 | AR 경로 편집 |
|------------|---------|------------|
| **지하철 운영사** | 역사 내 AR 경로 편집, 앵커 관리 | ✅ 핵심 |
| **병원/의료기관** | 원내 AR 길안내, 진료실 경로 | ✅ 확장 |
| **대형 쇼핑몰** | 매장 간 AR 경로, 프로모션 AR 배치 | ✅ 확장 |
| **식당/소상공인** | AR 메뉴판, QR 주문 | ⬜ 해당 없음 |

## 🗺️ 2. AR 경로 편집 모드 (AR Route Editor)

### 2.1 작업 워크플로

```
1. 공간 스캔          2. 노드 배치          3. 간선 연결          4. AR 배치         5. 테스트
   카메라로 역사         분기점·출구·게이트      노드 간 통로 연결      화살표·표지판·핀     모바일에서
   Point Cloud 추출     드래그&드롭 배치       거리·유형 설정        간선 위에 배치       즉시 QR 테스트
```

### 2.2 에디터 기능

| 기능 | 설명 |
|------|------|
| **스캔 모드** | USB 카메라 또는 연결된 모바일 카메라로 역사 내부 스캔 |
| **2D 플로어플랜 뷰** | 업로드된 도면 위에 노드·간선 오버레이 편집 |
| **노드 편집** | 드래그&드롭으로 분기점·출구·플랫폼·에스컬레이터 노드 배치 |
| **간선 편집** | 노드 간 통로 연결, 거리(m)·유형(복도/계단/에스컬레이터/엘리베이터) 설정 |
| **AR 오브젝트 배치** | 간선 위에 화살표·안내판·목적지 핀 배치 및 3D 미리보기 |
| **다국어 라벨** | AR 표지판에 en/ko/ja/zh/es/fr 번역 텍스트 입력 |
| **테스트 모드** | 편집한 경로를 모바일에서 즉시 테스트 (QR 코드 연결) |
| **앵커 라이프사이클** | 앵커 갱신 주기 관리, 만료 알림, 자동 갱신 설정 |

### 2.3 데이터 흐름

```
Partner Portal (Tauri V2)         Backend (Rust/Axum)           Tauri+Svelte App
    │                                  │                           │
    │── CreateNode mutation ─────────▶│                           │
    │── CreateEdge mutation ─────────▶│                           │
    │── PlaceARObject mutation ──────▶│                           │
    │                                  │── DB 저장 (PostGIS)       │
    │                                  │                           │
    │── 테스트 QR 생성 ─────────────▶│                           │
    │                                  │◀── indoorRoute query ────│
    │                                  │── AnchorSequence 리턴 ──▶│
```

## 🛡️ 3. 권한 관리

| 권한 레벨 | 대상 | 가능 기능 |
|----------|------|---------|
| **Super Admin** | ByteLogicCore 팀 | 모든 역사/공간 편집, 파트너 계정 관리 |
| **Station Admin** | 지하철 운영사 직원 | 담당 역사 AR 경로 편집, 앵커 관리 |
| **Partner Editor** | 병원/쇼핑몰 담당자 | 본인 시설 AR 경로 편집 |
| **Viewer** | 일반 파트너 직원 | 경로 조회·테스트만 가능 |

## 🔧 4. 기술 스택

| 레이어 | 기술 | 비고 |
|--------|------|------|
| **프레임워크** | Tauri V2 | Rust 기반 크로스플랫폼 데스크톱 |
| **프론트엔드** | React (Vite) | 에디터 UI, 플로어플랜 캔버스 |
| **캔버스** | React Flow / Konva.js | 노드·간선 그래프 시각 편집 |
| **3D 미리보기** | Three.js | AR 오브젝트 3D 렌더링 |
| **백엔드 통신** | GraphQL (기존 API 서버) | Mutation/Query 활용 |
| **카메라 스캔** | WebRTC + AR SDK bridge | USB/모바일 카메라 연결 |
