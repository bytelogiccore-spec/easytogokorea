## 1. 디자인 철학

| 원칙 | 설명 |
|------|------|
| **신뢰감** | 외국인 사용자가 낯선 환경에서도 안심하고 사용할 수 있는 UI |
| **입력 최소화** | 핵심 액션은 5터치 이내 완료, Zero-Keyboard 우선 (카드 선택, 슬라이더 위주) |
| **QR 중심** | 장소, 메뉴, 상품, 탑승권 등 모든 진입점 및 결과를 QR로 통일 |
| **현지감** | 한국의 아름다움을 UI에 반영 (전통 색상, 자연 이미지) |
| **범용성** | 노인부터 청년까지, 다양한 국적 사용자를 위한 직관적 UX |
| **사업자 편의** | 사업자(식당, 여행사 등) 앱은 다국어 자동 번역 등 반복 업무 제로화 목표 |

## 2. 컬러 시스템

### 주요 색상 팔레트

| 역할 | 색상명 | HEX | 용도 |
|------|--------|-----|------|
| Primary | Korean Blue | `#1A4B9B` | 주요 버튼, 링크, 강조 |
| Secondary | Hanji Cream | `#F5EDD6` | 배경, 카드 |
| Accent | Blossom Pink | `#E87F8D` | 알림, 하이라이트 |
| Success | Celadon | `#5DAD76` | 성공, 완료 상태 |
| Warning | Golden | `#E8A335` | 경고, 주의 |
| Error | Scarlet | `#DC3545` | 오류, 위험 |
| Neutral 900 | 거의 검정 | `#1A1A2E` | 주요 텍스트 |
| Neutral 500 | 회색 | `#6B7280` | 보조 텍스트 |
| Neutral 100 | 밝은 회색 | `#F9FAFB` | 배경 |

### 다크 모드

- 다크 모드 지원 (시스템 설정 연동)
- 야간 지도 탐색 시 자동 다크 모드 전환 옵션

## 3. 타이포그래피

| 역할 | 폰트 | 크기 | 굵기 |
|------|------|------|------|
| Display (타이틀) | Noto Sans KR | 32px | 700 |
| Heading 1 | Noto Sans KR | 24px | 600 |
| Heading 2 | Noto Sans KR | 20px | 600 |
| Body Large | Noto Sans | 18px | 400 |
| Body | Noto Sans | 16px | 400 |
| Caption | Noto Sans | 13px | 400 |
| Label | Noto Sans | 12px | 500 |

**폰트 선정 이유:** Noto Sans는 Google 제공 무료 폰트로, 한국어·중국어·일본어·아랍어·라틴어를 모두 커버합니다.

## 4. 컴포넌트 가이드

### 버튼

| 종류 | 배경 | 텍스트 | 용도 |
|------|------|--------|------|
| Primary | `#1A4B9B` | White | 주요 행동 (예약, 저장) |
| Secondary | Outlined | `#1A4B9B` | 보조 행동 |
| Destructive | `#DC3545` | White | 삭제, 취소 |
| Ghost | Transparent | `#6B7280` | 선택적 행동 |

### 카드 (장소, 투어, 병원)

```
┌────────────────────────────────────┐
│  [이미지 영역 - 160px]              │
├────────────────────────────────────┤
│  🏛️ 카테고리  ★ 4.8 (127 reviews)  │
│  **장소명 / 상품명**               │
│  📍 주소 또는 지역                  │
│  ⏰ 운영시간 또는 소요시간          │
│  💰 입장료 / 가격                   │
│                     [바로가기 →]   │
└────────────────────────────────────┘
```

### 채팅 말풍선

| 발신 | 수신 |
|------|------|
| 오른쪽 정렬, `#1A4B9B` 배경 | 왼쪽 정렬, `#F5EDD6` 배경 |
| 흰색 텍스트 | 어두운 텍스트 |
| 번역 표시 없음 | "[번역됨] 원문" 토글 표시 |

## 6. 다국어 지원 (i18n)

### 지원 언어

| 언어 | 코드 | 방향 |
|------|------|------|
| 영어 | `en` | LTR |
| 한국어 | `ko` | LTR |
| 중국어 (간체) | `zh-CN` | LTR |
| 일본어 | `ja` | LTR |
| 스페인어 | `es` | LTR |
| 프랑스어 | `fr` | LTR |
| 아랍어 | `ar` | RTL |

### i18n 파일 구조

```
locales/
├── en/
│   ├── common.json      # 공통 (버튼, 레이블, 에러)
│   ├── planner.json     # 일정 플래너
│   ├── travel.json      # 여행사 상담
│   └── medical.json     # 의료/보험
├── ko/
│   ├── common.json
│   └── ...
└── zh-CN/
    └── ...
```

### 번역 주의사항

- **동일 의미, 다른 표현**: 여행 관련 용어는 언어별 현지화 필수
  - 예: "itinerary" → 영어, "旅程" → 중국어, "日程" → 일본어
- **숫자 형식**: `1,000,000` (영어) vs `1,000,000` (한국어)  
- **날짜 형식**: `March 15, 2026` (영어) vs `2026년 3월 15일` (한국어)
- **통화 표시**: 사용자 국적에 따라 자동 변환

## 7. 접근성 (Accessibility)

WCAG 2.1 AA 기준을 목표로 합니다.

| 항목 | 요건 |
|------|------|
| 색상 대비 | 텍스트 최소 4.5:1 비율 |
| 터치 타깃 | 최소 44x44pt |
| 스크린 리더 | 모든 이미지 alt 텍스트 |
| 키보드 네비게이션 | 탭 순서 논리적 배치 |
| 오류 안내 | 색상만으로 표시 금지, 텍스트 함께 |

## 8. 모바일 최적화

### 상호작용 패턴

| 제스처 | 기능 |
|--------|------|
| 핀치 줌 | 지도 확대/축소 |
| 스와이프 | 일정 항목 삭제, 화면 전환 |
| 길게 누름 | 장소 핀 잡아서 이동 |
| 더블탭 | 지도 특정 위치 확대 |
| SOS 버튼 3초 누름 | 응급 모드 활성화 |

### 반응형 브레이크포인트

| 구간 | 범위 | 레이아웃 |
|------|------|---------|
| 모바일 | < 768px | 단일 컬럼 |
| 태블릿 | 768~1024px | 지도+패널 사이드바 |
| 데스크톱 | > 1024px | 3열 레이아웃 |

### 오프라인 UX

- 오프라인 상태 배너 표시
- 저장된 일정은 오프라인에서도 열람 가능
- 데이터 필요 기능은 회색 처리 + 설명 토스트
