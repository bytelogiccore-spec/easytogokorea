## 1. 프로젝트 소개

**EasyToGoKorea**는 한국을 방문하는 외국인 여행객을 위한 올인원(All-in-One) 여행 플랫폼입니다. 언어 장벽, 복잡한 대중교통 시스템, 의료 긴급 상황, 일정 관리의 어려움을 하나의 앱으로 해결하는 것을 목표로 합니다.

### 배경 및 문제 정의

| 문제 | 현황 | EasyToGoKorea 해결책 |
|------|------|----------------------|
| 언어 장벽 | 관광 안내판의 60% 이상이 영문 미지원 | 다국어 실시간 번역 및 안내 |
| 복잡한 이동 경로 | 버스·지하철·택시 통합 앱 부재 | 통합 대중교통 경로 탐색 |
| 의료 긴급 대응 | 외국인 보험 청구 서류 작성 어려움 | 자동화된 보험 청구 문서 생성 |
| 일정 계획 | 여행 정보 분산, 통합 플래너 없음 | 지도 기반 드래그&드롭 일정 관리 |
| 현지 전문가 부재 | 신뢰할 수 있는 여행사 연결 어려움 | 인증된 여행사 실시간 상담 |

## 2. 비전 및 목표

### 비전
> *"모든 외국인이 한국 여행 중 어떠한 불편함도 느끼지 않도록, 완벽한 여행 동반자를 제공한다."*

### 단기 목표 (6개월)
- MVP (최소 기능 제품) 출시
- 일정 플래너 + 공공 API 정보 제공 기능 완성
- 영어·중국어·일본어 지원
- 베타 사용자 1,000명 확보

### 중기 목표 (1년)
- 여행사 상담 플랫폼 런칭 (파트너 여행사 50개 이상)
- 의료 보험 청구 기능 완성
- 월간 활성 사용자(MAU) 10만 명 달성

### 장기 목표 (2년+)
- 동남아시아·일본 등 타 국가 여행 플랫폼으로 확장
- AI 기반 개인화 여행 추천 시스템 고도화
- 보험사·병원·여행사와의 공식 파트너십 확대

## 3. 핵심 기능 개요

### Feature 1 - 지도 기반 일정 플래너 🗺️
- 인터랙티브 지도 위에서 여행지 핀 추가, 드래그&드롭 순서 변경
- 여행지 간 최적 경로 자동 계산 (Valhalla 라우팅)
- 일정 공유 및 협업 편집 / 오프라인 지도 지원
- 템플스테이·축제·공연 장소 별도 레이어

### Feature 2 - 공공 API 정보 허브 📡
- 한국관광공사 관광 명소·행사·전통시장 정보
- 실시간 대중교통 (버스·지하철·KTX), 날씨·환율
- 문화재청 유적지, 한국불교문화사업단 템플스테이 API

### Feature 3 - 여행 전 문화 팁 가이드 📖
- 한국 문화 상식 카드뉴스 (신호등 바닥 LED 등 9개 카테고리)
- 위치 기반 상황별 알림 (지하철역·식당·관광지 진입 시)
- 긴급 번호 (112 / 119 / 1330) 원터치

### Feature 4 - 문화 체험 & 공연 플랫폼 🏯🎤
- 템플스테이·전통공예·한방·역사 투어 예약
- K-POP 신인/인디 아티스트 공연 검색·예약·후원
- K-드라마 성지 지도 레이어, 팬미팅 소규모 이벤트

### Feature 5 - 축제 정보 & 셔틀 버스 🎆🚌
- 연 1,000개+ 지역 축제 다국어 안내·캘린더
- 호텔↔축제장 왕복 셔틀 버스 좌석 예약 (QR 탑승권)

### Feature 6 - 식당 메뉴판 다국어 & 결제 🍽️
- 테이블 QR 스캔 → 6개 언어 메뉴 즉시 표시
- 앱 내 외화 결제 (Visa/UnionPay/알리페이/WeChat Pay)
- 할랄·비건 필터, 알레르기 태그

### Feature 7 - 지역 특산품 직구 마켓 🛍️
- 시장 QR 스캔 → 상품 다국어 페이지 + 즉시 결제
- 해외 직배송 / 묶음 배송 (배송비 절감)
- 지역 정품 인증 마크

### Feature 8 - 의료 보험 & 생활 서비스 🏥
- My HealthWay / HIRA API 연동 보험 서류 자동화
- 외국인 특화 병원 검색·예약, 실시간 통역 연결
- 비자 안내, 외국인등록증, 장기 체류 지원

### Feature 9 - 소셜 여행 커뮤니티 피드 📸
- 인스타그램 방식의 실시간 여행 후기 및 사진 공유 (피드)
- 장소 태그 기반으로 내 일정에 장소 즉시 추가 기능
- 다국어 댓글 번역 및 사용자/크리에이터 소셜 팔로우

### 마이크로서비스 구성

> **점진적 MSA 전략**: MVP는 모듈러 모놀리스로 시작, MAU 5만 초과 시 병목 도메인부터 분리.
> 의료 서비스는 규정 준수를 위해 처음부터 독립 배포.

| 서비스 | 역할 | 기술 | 포트 |
|--------|------|------|------|
| `auth-service` | 사용자 인증·인가, JWT 발급 | Node.js + Express | 3001 |
| `planner-service` | 일정 CRUD, 지도·경로 처리 | Node.js + Express | 3002 |
| `info-service` | 공공 API 수집·캐싱 (관광/날씨/환율/축제/팁) | Node.js + Express | 3003 |
| `chat-service` | 실시간 채팅, 여행사·통역 매칭 | Node.js + Socket.IO | 3004 |
| `medical-service` | 병원 예약, 보험 서류, 통역 연결 | Node.js + Express | 3005 |

## 5. 사용자 흐름 (User Flow)

### 신규 사용자 온보딩
```
앱 설치 → 언어 선택 → 회원가입/소셜로그인 → 여행 정보 입력
(여행 날짜, 관심사, 예산) → 맞춤 추천 일정 제안 → 일정 편집 → 여행 시작
```

### 일정 플래너 흐름
```
지도 열기 → 장소 검색/추천 탐색 → 장소 일정에 추가 →
경로 최적화 → 이동 수단 선택 → 일정 확정 → (선택) 공유
```

### 여행사 상담 흐름
```
상담 요청 → 여행사 목록 탐색 (필터: 언어/전문분야/평점) →
여행사 선택 → 실시간 채팅 시작 → 투어 상품 확인 →
예약 및 결제 → 확정 알림 수신
```

### 의료 보험 청구 흐름
```
병원 도착 → 앱에서 병원 검색 → 예약 또는 바로방문 등록 →
진료 후 영수증 촬영 (OCR) → 보험 청구 서류 자동 생성 →
서류 PDF 다운로드/이메일 전송 → 보험사 제출
```

## 6. 주요 이해관계자

| 이해관계자 | 역할 | 주요 관심사 |
|-----------|------|------------|
| **외국인 여행객** | 앱 최종 사용자 (완전 무료) | 편의성, 신뢰성, 입력 최소화 |
| **여행사·문화 가이드** | 투어·체험 제공자 | 고객 유입, 예약 전환율 |
| **사찰·문화기관** | 템플스테이·체험 제공자 | 외국인 예약 증가, 홍보 |
| **인디·신인 아티스트** | 공연 등록자 | 글로벌 팬 연결, 수익 |
| **지역 특산품 생산자** | 마켓 판매자 | 해외 판로 확대 |
| **소규모 식당** | 메뉴 QR 사업자 | 외국인 응대 편의, 결제 |
| **지역 버스 업체** | 셔틀 운행사 | 안정적 수요, 정산 |
| **병원·의료기관** | 의료 서비스 제공자 | 외국인 환자 유치 |
| **보험사** | 데이터 수신자 | 표준화된 청구 서류 |
| **지방자치단체** | 관광 정책 파트너 | 지역 관광 활성화 |
| **한국관광공사** | 데이터·홍보 파트너 | API 활용, 공동 마케팅 |
| **ByteLogicCore** | 개발·운영사 | B2B 수수료·구독 수익 |
