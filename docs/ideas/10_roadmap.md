## 2. 이해관계자 맵

| 이해관계자 | 역할 | 주요 관심사 |
|-----------|------|-----------|
| **외국인 여행객** | 최종 사용자 | 편의성, 신뢰성, 언어 지원 |
| **사찰 / 문화기관** | 문화 체험 제공자 | 외국인 예약 증가, 홍보 |
| **인디·신인 아티스트** | 공연 등록자 | 글로벌 팬 연결, 수익 기회 |
| **지역 특산품 생산자** | 마켓 판매자 | 판로 확대, 해외 수출 |
| **소규모 식당** | 메뉴 QR 사업자 | 외국인 고객 응대 편의 |
| **여행사 파트너** | 투어 상품 제공자 | 예약 전환율, 고객 유입 |
| **의료기관** | 의료 서비스 제공자 | 외국인 환자 유치 |
| **지역 버스 업체** | 셔틀 버스 운행 | 안정적 수요, 정산 편의 |
| **지방자치단체** | 관광 정책 파트너 | 지역 관광 활성화 |
| **한국관광공사** | 데이터 / 홍보 파트너 | API 활용, 공동 마케팅 |
| **ByteLogicCore** | 개발·운영사 | 수익 창출, 플랫폼 성장 |
