# 07. 지역 특산품 직구 마켓

> **상위 목차**: [ideas/00_index.md](00_index.md)

---

## 배경 및 기회

외국인 여행객이 현지에서 마음에 드는 지역 특산품을 발견해도
무거운 짐 문제, 언어 장벽, 해외 배송 불가로 구매를 포기하는 경우가 많음.
**"여행 중 발견 → 앱에서 해외 배송 주문"** 으로 전환하는 수요가 큰 틈새 시장.

---

## 1. 핵심 기능

| 기능 | 설명 |
|------|------|
| **지역별 특산품 탐색** | 지도 위에 특산품 마켓 핀 표시 (보성 녹차, 이천 도자기, 제주 한라봉 등) |
| **현장 QR 스캔 구매** | 시장/상점의 QR 스캔 → 상품 다국어 정보 + 즉시 결제 |
| **해외 직접 배송** | 여행 후 귀국해도 집으로 배송받을 수 있는 국제 배송 옵션 |
| **대량 묶음 배송** | 같은 여행지에서 산 물건 모아 한 번에 배송 (배송비 절감) |
| **정품 인증** | 지역 명품 인증 마크 (지자체 협력) — 가짜 제품 방지 |

---

## 2. 판매자 등록 플로우 (사업자 앱)

```
판매자 등록 → 상품 등록 (한국어 입력)
→ 자동 다국어 번역 (Papago/DeepL API)
→ QR코드 자동 생성
→ 주문 알림 수신 → 배송 대행사 연결 → 정산
```

---

## 3. 사용자 UX (최소 입력)

- 상품 검색 없이 **QR 스캔 한 번** → 상품 페이지 진입
- 배송 주소는 **첫 구매 시 1회 등록 후 자동 적용**
- 결제는 등록 카드 원터치

---

## 4. 수익 모델

| 항목 | 수수료 |
|------|-------|
| 거래 중개 수수료 | 판매금액의 5~8% |
| 국제 배송 중개 마진 | 건당 2,000~5,000원 |
| 프리미엄 상품 상단 노출 | 월정액 |
| 사용자 입장 | **무료** (거래 수수료는 판매자 부담) |
