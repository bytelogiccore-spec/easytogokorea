# 03. 여행 전 문화 팁 가이드

> **상위 목차**: [ideas/00_index.md](00_index.md)

## 배경 및 기회

외국인 여행객이 한국에서 겪는 당혹스러운 순간 대부분은 **사전에 몰랐기 때문**에 발생한다.
구글링으로 알 수 있지만 한 곳에 모아 "읽기 쉽게, 미리 알려주면" 여행 경험 만족도가 크게 오른다.
여행 전 설레는 시점에 앱을 열어볼 이유를 만들어주는 효과도 있다 → **앱 설치 전환율 상승**.

## 콘텐츠 구성 — 카테고리별 팁

| 카테고리 | 예시 팁 |
|---------|--------|
| 🚦 **교통·길거리** | 신호등 바닥 LED 경고등 / 대기선 밟지 않기 / 지하철 노약자석 문화 |
| 🍽️ **식당 예절** | 반찬은 무한 리필 공짜 / 음식 남겨도 됨 / 혼밥 문화 자연스러움 |
| 💳 **결제·쇼핑** | 현금 거의 불필요 / 편의점 계산대 바구니 올리기 / 영수증 자동 발급 |
| 🙇 **인사·예절** | 두 손으로 건네기 / 나이 물어보는 문화 / 어른 앞에서 술잔 돌리기 |
| 🏨 **숙박** | 체크아웃 전 쓰레기 분리수거 / 슬리퍼 있는 곳 신발 벗기 |
| 🚿 **목욕·웰니스** | 찜질방 남녀 공용 공간 / 때밀이 문화 / 대중탕 에티켓 |
| 📵 **디지털·통신** | 공공 Wi-Fi 이용법 / 네이버 vs 구글 현지 차이 / T-money 카드 |
| 🌸 **계절·날씨** | 황사 시즌 파악 / 장마철 주의 / 축제 시즌 숙박 사전 예약 필수 |
| 🚨 **긴급 상황** | 112 (경찰) / 119 (소방·응급) / 1330 (관광 다국어 안내) <br> **💡 팁:** 길을 잃었을 때 가로등 QR코드나 전봇대 노란 번호를 앱에 입력하면 내 위치를 바로 알 수 있어요! |

## 사용자 경험

**여행 전 (앱 설치 시점)**
```
온보딩 완료 → "한국 여행 꿀팁 10가지" 카드 피드 자동 제공
 → 카드 스와이프 방식 (한 화면에 팁 하나)
 → "저장" 누르면 오프라인 북마크
```

**여행 중 (위치 기반 상황별 알림)**
```
[지하철역 접근 시] "노약자석(분홍색 좌석)은 비워두세요 🙏"
[식당 QR 스캔 시]  "이 식당은 반찬 무한 리필 가능해요!"
[관광지 입장 시]   "사진 촬영 가능 구역 / 불가 구역 안내"
[찜질방 근처 시]   "찜질방 입장 방법 안내 보기"
```

**콘텐츠 형식**
- 카드뉴스형 짧은 텍스트 + 일러스트 이미지
- 영상 클립 (15초 이내 — 신호등 LED, 찜질방 입장법 등 직접 시연)
- 다국어 제공 (영어·중국어·일본어·스페인어 등)

---

## 수익 연결

- 팁 콘텐츠 자체는 전부 **무료**
- 지자체·한국관광공사 공동 콘텐츠 제작 → **관광 홍보 협력 수익**
- 팁 카드 내 연관 서비스 자연 연결 (찜질방 팁 → 근처 찜질방 예약 버튼)
