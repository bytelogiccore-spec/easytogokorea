## 1. 플랫폼 비전

### 핵심 비전
> *"한국의 모든 콘텐츠 — 관광, 문화 체험, 음식, 쇼핑, 의료, 이동 — 를 언어·문화 장벽 없이 이용할 수 있는 통합 생활 플랫폼"*

### 핵심 가치 (Core Values)

| 가치 | 설명 |
|------|------|
| **접근성** | 언어·기술 장벽 없이 누구나 쉽게 |
| **빠름** | 최소 입력, 최대 처리 속도 |
| **신뢰성** | 검증된 파트너, 안전한 결제 |
| **현지성** | 대형 플랫폼이 못 하는 로컬 밀착 서비스 |

## 2. 앱 구분 전략

### 2-1. 사용자(여행객) 앱 — **모바일 전용**

```
타겟: 외국인 관광객 / 장기 체류자
플랫폼: Desktop/Mobile (Tauri V2 + Svelte)
특징:
  - 설치 즉시 언어 자동 감지 → 원클릭 온보딩
  - 입력 최소화 UX (카드형 터치 선택, QR 스캔 중심)
  - 오프라인 핵심 기능 지원 (지도, 메뉴 번역, 비상 연락)
  - 여행 시작~종료까지 원스톱 플로우
```

**핵심 온보딩 플로우 (3단계)**
```
① 언어 선택 (자동 감지, 1터치 확인)
② 여행 기간·목적 선택 (카드 탭 — 관광 / 문화체험 / 쇼핑 / 의료 / 복합)
③ 추천 일정·콘텐츠 즉시 제공
→ 총 입력 항목: 3개 이하
```

**Bottom Navigation 구성**
```
홈 / 지도 / [QR스캔 — 중앙 고정] / 마켓 / 내 여행
```

### 2-2. 사업자 앱 — **모바일 + 윈도우 (Tauri 기반)**

```
타겟: 여행사, 식당, 사찰/문화기관, 특산품 판매자, 의료기관
플랫폼: Windows (Tauri), Desktop/Mobile (Tauri V2 + Svelte)
특징:
  - 예약 관리 대시보드
  - 상품/메뉴 등록 및 다국어 자동 번역 (Papago/DeepL API)
  - 매출 집계 및 정산
  - QR코드 메뉴판 자동 생성
  - 엑셀/CSV 대량 업로드 지원
```

**사업자 타입별 기능**

| 사업자 유형 | 주요 기능 |
|------------|---------|
| 여행사 | 투어 상품 등록, 실시간 상담, 예약 관리 |
| 식당 | 다국어 메뉴 등록, QR 주문, 외화 결제 |
| 사찰/문화기관 | 템플스테이 예약 캘린더, 프로그램 관리 |
| 특산품 판매자 | 상품 등록, 해외 배송 설정, 주문 관리 |
| 의료기관 | 외국인 예약 관리, 통역 매칭, 서류 관리 |
| 공연장·아티스트 | 공연 일정 등록, 티켓 판매, 정산 |
| 축제·버스업체 | 셔틀 노선 등록, 좌석 관리, 실시간 위치 공유 |
