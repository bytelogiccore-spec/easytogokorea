# 05. 축제 정보 & 셔틀 버스 패키지

> **상위 목차**: [ideas/00_index.md](00_index.md)

---

## 배경 및 기회

한국의 지역 축제는 화천 산천어 축제부터 보령 머드 축제까지 **연 1,000개 이상**이지만,
외국인에게는 정보가 묻혀 있고 **현장까지 가는 방법이 큰 장벽**이다.
특히 대중교통으로 접근하기 어려운 지방 축제는 택시비 부담과 음주 후 귀환 문제도 있다.
**축제 정보 + 셔틀 버스를 세트 상품으로 묶으면** 단독 구매 수요가 높은 킬러 상품이 된다.

---

## 1. 축제 정보 허브

| 콘텐츠 | 세부 내용 |
|---------|---------|
| **축제 디렉토리** | 지역·계절·테마별 필터링 (불꽃 / 벚꽃 / 눈·얼음 / 음식 / 음악 등) |
| **다국어 상세 안내** | 스케줄·지도·입장료·주의사항 / 대표 음식·체험 안내 |
| **날씨·복장 팁** | 현장 날씨 예보 연동, 추천 복장·준비물 가이드 |
| **연간 축제 캘린더** | 다음 여행 일정을 잡을 때 참고할 12개월 달력 제공 |
| **실시간 현황** | 혼잡도 / 셔틀 잔여석 마감 / 특이 상황 알림 |

**파트너십 우선순위**: 화천 산천어축제 / 진해 군항제(벚꽃) / 보령 머드축제 / 함평 나비축제 / 부산 불꽃축제 등 **외국인 유입이 많은 대형 축제** 중심.

---

## 2. 셔틀 버스 서비스

```
[여행 일정에 축제 관심 등록]
        ↓
[축제 선택] → 셔틀 버스 노선 자동 표시 (출발지·시간·잔여석)
        ↓
[좌석 구매] 출발지 (서울 주요 거점 / 인천공항 등) → 축제 현장 왕복
        ↓
[QR 탑승권] 발급 + 실시간 버스 위치 확인
```

**출발지 유형**

| 유형 | 설명 |
|------|------|
| 서울 주요 거점 (강남역·명동·홍대) | 외국인 숙박 밀집 지역 기준 |
| 인천공항 경유 | 입국 당일 축제로 바로 이동 가능 |
| 고속버스 터미널 경유 | 지방 축제 접근 |

---

## 3. 사용자 플로우 (2터치 입력)

```
① 홈 "축제" 탭 터치
② 내 여행 기간 기준 추천 축제 3개 카드 자동 표시
③ 축제 터치 → 상세 (사진, 특산물, 다국어 안내)
④ "셔틀 버스 예약" 버튼 터치
⑤ 출발지 선택 · 인원 선택 · 결제 (원터치)
⑥ QR 탑승권 발급 + 실시간 버스 위치 확인
→ 총 입력 항목: 2개 (출발지, 인원)
```
