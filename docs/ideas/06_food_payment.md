# 06. 식당 메뉴판 다국어 & 간편 결제

> **상위 목차**: [ideas/00_index.md](00_index.md)

---

## 배경 및 기회

외국인이 한국 식당에서 경험하는 가장 큰 불편:
**메뉴판 이해 불가 + 현금 결제 강요 + 팁 문화 혼란**.
식당 측도 외국인 응대가 어려운 상황.
양쪽의 문제를 **테이블 QR 하나**로 해결한다.

---

## 1. 사용자 경험 플로우

```
① 테이블에 부착된 QR 코드 스캔 (앱 or 카메라)
② 메뉴판 즉시 표시 — 사용자 언어로 자동 번역
③ 사진·재료·알레르기 정보 확인
④ 주문 항목 선택 (수량 터치)
⑤ 앱 내 바로 결제 (카드 / 외화 결제 / 환율 자동 표시)
⑥ 주방 주문서 자동 전달 (사업자 앱·POS 연동)
→ 종이 메뉴판 필요 없음, 직원과의 소통 최소화
```

---

## 2. 식당 사업자 기능 (사업자 앱 / Windows 앱)

| 기능 | 설명 |
|------|------|
| **메뉴 등록** | 한국어로 입력 → 자동 번역 (Papago/DeepL API — 6개 언어) |
| **사진 첨부** | 메뉴 사진 1장 업로드로 완성 |
| **알레르기 태깅** | 주요 알레르기 항목 체크박스 선택 |
| **할랄·비건 표시** | 메뉴별 인증 마크 태그 |
| **QR 코드 생성** | 테이블별 QR 자동 생성 & 출력 |
| **주문 수신** | 실시간 주문 알림 + 프린터 연동 |
| **매출 현황** | 외국인 / 내국인 매출 분리 집계 |

---

## 3. 결제 처리

| 결제 수단 | 처리 방법 |
|---------|---------|
| 신용카드 (Visa, Master, UnionPay) | Stripe / Portone 연동 |
| Apple Pay / Google Pay | OS 네이티브 결제 연동 |
| 알리페이 / WeChat Pay | 중국 관광객 대응 |
| 외화 자동 환산 | 실시간 환율 표시 (한국은행 API) |

---

## 4. 추가 가치

- 식당 리뷰 (외국인 전용) — 앱에서 직접 별점·텍스트 리뷰 작성
- 할랄 인증, 비건 메뉴 필터링
- "이 메뉴 일정에 추가" → 일정 플래너 연동

---

## 5. 수익 모델

| 항목 | 내용 |
|------|------|
| 식당 결제 중개 수수료 | 결제금액의 1.5~2.5% |
| 사업자 구독 (SaaS) | 월 30,000원 (QR 메뉴 시스템 이용료) |
| 사용자 입장 | **무료** |
