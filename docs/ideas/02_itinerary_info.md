## 1. 지도 기반 일정 플래너 🗺️

- 인터랙티브 지도에서 핀 추가, 드래그&드롭 순서 변경
- 여행지 간 최적 경로 자동 계산 (Valhalla 라우팅 엔진)
- 일정 공유 및 협업 편집
- 오프라인 지도 다운로드 지원 (전 지역, 무료)
- 템플스테이·문화 체험 장소 별도 레이어 추가
- **AR 도보 네비게이션**: 전봇대/가로등 QR 스캔 시, 현재 위치를 기점으로 카메라 화면에 증강현실(AR) 화살표를 띄워 목적지까지 직관적인 길안내 제공.

## 2. 공공 API 정보 허브 📡

| 카테고리 | 정보 | API 출처 |
|---------|------|---------|
| 관광 명소 | 전국 관광지 정보, 사진, 운영시간 | 한국관광공사 TourAPI 4.0 |
| 문화 체험 | 전통 체험 프로그램 목록 | 한국문화정보원 KCISA |
| 역사 유적 | 유적지 정보, 해설 콘텐츠 | 문화재청 국가문화유산포털 |
| 대중교통 | 버스·지하철·KTX 실시간 정보 | 공공데이터포털 |
| 날씨·대기질 | 기상 예보, 미세먼지 | 기상청 API |
| 환율 | 실시간 환율 | 한국은행 API |
| 지역 축제·행사 | 전국 행사 정보 | 한국관광공사 |
| 전통시장 | 전국 전통시장 위치·운영 정보 | 소상공인시장진흥공단 |
| 공공 화장실·관광안내소 | 위치 정보 | 공공데이터포털 |

## 3. 여행사 상담 플랫폼 💬

- 인증된 여행사와 실시간 채팅
- 다국어 통역 지원 상담사 매칭
- 투어 상품 예약 및 결제
- 여행사 리뷰 및 평점 시스템
- 맞춤형 여행 상품 요청

**전문 가이드 유형 확장**

| 가이드 유형 | 전문 분야 |
|-----------|---------|
| 사찰 전문 가이드 | 템플스테이, 불교 문화 |
| 한복·공예 체험 | 전통 문화 체험 |
| 역사 유적 | 궁궐, 세계유산 해설 |
| K-콘텐츠 | K-드라마 촬영지, K-POP 성지 |
| 의료 관광 | 한방 웰니스, 성형·미용 |
