# 13. 온디바이스 AI 및 라우팅 제어 (Android)

## 1. 개요
Android 기기의 내장 AI 모델(Gemini Nano)과 시스템 API(App Actions, ML Kit)를 활용하여 `EasyToGoKorea` 앱 내 서비스의 자동 라우팅, 명령 수행, 그리고 오프라인 번역 기능을 제공하는 기술 명세입니다. 서버 통신 없이 기기 내부에서 처리되므로 오프라인 환경에서도 빠른 응답 속도와 프라이버시 보호를 보장합니다.

## 2. 핵심 기술 스택
### 2.1. App Actions 및 Deep Linking
*   **목적**: Google Assistant의 음성/텍스트 명령을 앱 내부의 특정 뷰(화면)나 기능으로 직접 매핑.
*   **방식**: `shortcuts.xml`을 통한 인텐트(Intent) 정의 및 딥링크 URI(예: `easytogokorea://feature/planner`)로 파라미터 전달. 사용자가 발화한 위치, 날짜 등의 변수를 앱 로딩 시 함께 넘겨줍니다.

### 2.2. Gemini Nano (Android AICore)
*   **목적**: 기기 내장형 초경량 LLM을 활용한 자연어 이해(NLU), 의도 파악 및 텍스트 요약.
*   **방식**: 사용자가 앱의 검색창이나 챗봇에 입력한 모호한 문장("나 지금 한국 문화 체험하고 싶은데 추천해줘")을 시스템 연산을 통해 구조화된 라우팅 명령(`{"action": "search", "category": "culture", "location": "current"}`)으로 변환하여 해당 화면으로 즉시 스위칭합니다.

### 2.3. ML Kit On-device Translation
*   **목적**: 네트워크 연결이 없거나 데이터 로밍이 불안정한 상황에서도 매끄럽게 작동하는 실시간 텍스트 및 이미지 번역 기능.
*   **방식**: Google ML Kit의 On-device Translation API를 활용합니다. 사용자가 필요한 언어팩(예: 영어-한국어)을 최초 1회 다운로드(약 30MB 수준)해 두면, 이후에는 기기 로컬에서 텍스트 감지 및 번역을 초저지연으로 처리합니다.

## 3. 주요 유스케이스
1.  **AI 음성 명령 라우팅**: "오케이 구글, EasyToGoKorea에서 내일 서울 여행 일정 보여줘" -> 플래너 화면(`planner.html`)으로 앱 자동 실행, '내일, 서울' 필터 실시간 적용.
2.  **인텐트 기반 화면 자동 전환**: 앱 내 검색창에 "배가 아파서 약국 가야해" 작성 -> 문장 의도 분석 후 의료/생활 지원 화면(`08_medical_living.md` 관련 뷰)으로 즉시 라우팅.
3.  **실시간 오프라인 메뉴/간판 번역 자동화**: 식당 메뉴 화면이나 AR 가이드 화면에서 카메라를 켜면 -> ML Kit을 통해 서버 지연 없이 뷰파인더 위에 한국어 메뉴/간판을 외국인 사용자의 모국어로 실시간 오버레이 번역. 화면 스위칭 없이 동작.

## 4. 지원 가능한 기기 목록 (Android)
온디바이스 AI 기능은 기기의 NPU/TPU 연산 능력과 OS 버전에 영향을 받습니다. 기능별로 지원 범위가 나뉩니다.

*   **Gemini Nano (AICore) 지원 최고급 하드웨어 연산 기기 (자연어 이해 라우팅)**:
    *   **Google**: Pixel 8 Pro, Pixel 8, Pixel 8a (개발자 옵션), Pixel 9 시리즈 전 모델
    *   **Samsung**: Galaxy S24, S24+, S24 Ultra, Galaxy Z Fold6, Z Flip6
    *   **기타**: Motorola Edge 50 Ultra 등 퀄컴 Snapdragon 8 Gen 3 이상 또는 최신 텐서 칩셋 탑재 플래그십 기기
*   **ML Kit 실시간 오프라인 번역 및 기초 App Actions 지원 기기**:
    *   **OS 요구사항**: Android 5.0 (API Level 21) 이상.
    *   **호환성**: 사실상 **현재 시장에 있는 95% 이상의 거의 모든 안드로이드 스마트폰**. 언어 모델 패키지만 앱 내에서 다운로드 받으면 저사양 기기에서도 실시간 오프라인 번역과 기초적인 딥링크 연동은 원활하게 구동됩니다.
