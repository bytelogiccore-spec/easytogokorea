## 1. 템플스테이 & 한국 전통 문화 체험 🏯

### 배경
한국 템플스테이는 연간 **50,000명 이상** 외국인 참가자를 보유하며 지속 성장 중이나,
영어 기반 예약 시스템이 부재하거나 복잡한 상황.

### 체험 카테고리

| 카테고리 | 세부 내용 | 주요 파트너 |
|---------|---------|-----------|
| **템플스테이** | 1박 2일~장기 수행, 발우공양, 명상 프로그램 | 한국불교문화사업단 (조계종) |
| **전통 문화 체험** | 한복 체험, 도자기·한지 공예, 전통 음악(국악) | 지역 문화원, 체험 공방 |
| **K-뷰티 / K-웰니스** | 한방 치료, 찜질방, 명상 센터 | 한의원, 웰니스 센터 |
| **역사 유적 투어** | 경복궁, 해인사, 첨성대 등 UNESCO 세계유산 | 문화재청 API 연동 |
| **로컬 마을 체험** | 민속마을, 어촌·농촌 체험 | 농어촌공사 협력 |

### 예약 플로우 (4터치 이내)
```
① 홈 카드 "문화 체험" 탭 터치
② 지역 또는 테마 선택 (카드형 — 템플스테이 / 전통공예 / 한방 / 역사투어)
③ 날짜·인원 선택
④ 결제 (등록 카드 원터치 또는 Google/Apple Pay)
⑤ 확정 알림 + 오프라인 가이드 자동 다운로드
```

### API 연동

| API / 데이터 소스 | 내용 |
|----------------|------|
| 한국불교문화사업단 템플스테이 API | 사찰 목록, 프로그램, 예약 가능 일정 |
| 문화재청 국가문화유산포털 API | 유적지 정보, 해설 콘텐츠 |
| 한국문화정보원 KCISA | 전통 체험 프로그램 목록 |
| 한국관광공사 TourAPI 4.0 | 문화시설, 행사 정보 |

## 2. K-POP 신인 아티스트 공연 플랫폼 🎤

### 배경
빅4 유명 아이돌 외에 **인디·신인·비주류 아티스트**의 공연 정보는 외국인이 접하기 매우 어렵다.
신인 아티스트 입장에서도 **외국인 팬과 연결될 채널**이 없다.

> 외국인 팬: "숨겨진 아티스트를 현장에서 먼저 발견"하는 경험
> 신인 아티스트: 글로벌 팬과의 접점 + 실제 수익 기회

### 핵심 기능

| 기능 | 설명 |
|------|------|
| **공연 검색 & 예약** | 다국어 지원 일정. **지자체 지원/공공 유휴 공연장** 활용으로 대관료 최소화 |
| **아티스트 프로필** | 음악 미리듣기, SNS 연결 |
| **팬미팅 소규모 이벤트** | 10~50명 규모 팬미팅 (보드게임 카페, 빈 유휴 공간 활용) |
| **팝업 이벤트 (Pop-up)** | K-POP 랜덤 댄스, 버스킹 등 게릴라성 야외 이벤트 (실시간 위치 기반 알림 및 참여 유도) |
| **공연 알림** | 관심 아티스트 등록 → 여행 기간 내 공연 푸시 알림 |
| **자율 관람료 & 현장 굿즈** | 기본 티켓은 **초저가 또는 무료**. 공연 후 감동만큼 "팁(Tip) 후원" 결제 및 앱 내 QR로 아티스트 굿즈(현장 수령) 구매 |
| **현장 영상 공유** | 공연 후 짧은 클립 (아티스트 동의 하에) → SNS 바이럴 |

### 사용자 플로우
```
① 홈 "K-POP 공연" 탭 터치
② 내 여행 기간 자동 필터링 → 기간 내 공연 목록 표시
③ 아티스트 프로필 확인 (음악 미리듣기, SNS 링크)
④ 티켓 선택 + 결제 (원터치)
⑤ QR 티켓 발급 → 현장 입장
→ 총 입력 항목: 2개 이하 (여행 기간은 온보딩에서 이미 입력)
```

### 수익
- **대관료 제로 전략**: 국가 및 지자체 지원 사업(유휴 공간 활용) 연계로 아티스트의 초기 비용 제거
- **티켓/후원 수수료**: 티켓 무료 발급 시 수수료 0원. 관객이 지불하는 "자율 후원금(Tip)"의 **5~8%** 수취
- **굿즈 판매 수수료**: 현장에서 판매되는 아티스트 굿즈(포토카드, 앨범 등) 판매 금액의 **5~8%** 수취
- 사용자 입장: 기본 관람은 매우 저렴하거나 **무료** (자발적 후원 유도)

## 3. K-드라마 / K-팝 성지 투어 🎬

- 드라마 촬영지, 아이돌 굿즈샵, 팬미팅 장소 지도 레이어
- 팬미팅 일정 연동 (위 K-POP 공연 플랫폼과 연결)
- 팬 커뮤니티 공유 기능 (같은 팬들과 동선 공유)
- **성지 방문 스탬프 랠리 (게이미피케이션)**: 특정 성지 주변의 가로등/전봇대 QR을 스캔하면 앱 내에 디지털 스탬프 획득. 배지 컬렉션 완성 시 포인트 또는 주변 카페 할인 쿠폰 지급.

## 4. 교통 통합 패스 🚇

- 대중교통 + 관광지 입장권 묶음 패스 (Korea Tourism Pass 연계)
- 앱 내 NFC/QR로 개찰구 통과
- 1일권 / 3일권 / 7일권 옵션
