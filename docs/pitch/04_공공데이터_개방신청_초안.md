- **신청 유형**: 데이터 개방 신청 (신규)

### 개방 요청 데이터

- **대상 기관**: 농림축산식품부 (공공배달 통합포털 운영)
- **요청 데이터명**: 전국 공공배달앱 가맹점 메뉴 및 가격 정보
- **요청 데이터 항목**:
  1. 가맹점 고유 ID
  2. 가맹점명 (한글)
  3. 업종 분류 (한식/중식/일식/양식/카페 등)
  4. 메뉴명 (한글)
  5. 메뉴 가격 (원)
  6. 메뉴 설명 (한글)
  7. 메뉴 이미지 URL (있을 경우)
  8. 소속 공공배달앱명 (배달특급/먹깨비/땡겨요 등)
  9. 주문 가능 URL 또는 딥링크 (해당 가맹점의 공공배달앱 주문 페이지 직접 링크)
  10. 결제 수단 정보 (제로페이/지역화폐/카드 가능 여부)
- **요청 데이터 형식**: REST API (JSON) 또는 CSV 정기 업데이트
- **연계 데이터**: 현재 data.go.kr에 공개된 "배달특급 가맹점 현황" 데이터에 메뉴 정보가 누락되어 있어 보완 요청

### 활용 목적

```
방한 외국인 관광객(연간 2,500만 명) 대상 다국어 음식 정보 서비스 개발에 
활용하고자 합니다.

현재 외국인 관광객이 한국 식당의 메뉴를 이해하려면 네이버 지도 → 
한국어 검색 → 메뉴판 사진 확인이라는 복잡한 과정을 거쳐야 합니다.
공공배달앱 가맹점의 메뉴 데이터가 개방되면, AI 자동 번역을 통해 
외국인이 한국어를 모르더라도 지역 맛집의 메뉴와 가격을 즉시 
확인할 수 있는 서비스를 구축할 수 있습니다.

이는 한국관광공사(KTO)의 2026년 관광 수입 30조 원 목표와 
농림축산식품부의 K-푸드 세계화 정책에도 부합합니다.

또한, 해당 데이터 개방은 귀 기관의 공공데이터 활용 실적으로 
직접 반영될 수 있으며, 서비스 출시 후 데이터 활용 사례로 
보고 가능합니다.
```

### 기대 효과
1. 외국인 관광객의 한국 음식 접근성 향상
2. 공공배달앱 가맹 소상공인의 외국인 고객 유치 기회 확대
3. 공공데이터 활용 스타트업 육성 성과 창출
4. K-푸드 홍보 효과
5. 공공배달앱의 외국인 이용자 확대 (신규 고객층 유입)

### 향후 협력 제안: 주문·결제 API 연동

본 데이터 개방 요청과 별도로, **서비스 연동 파트너십**을 제안드립니다.

메뉴 데이터 활용을 통해 외국인 이용자 트래픽이 확인되면,
다음 단계로 공공배달앱의 **주문·결제 시스템 연동**을 협의하고자 합니다.

#### 단계별 연동 로드맵

| 단계 | 연동 방식 | 필요 사항 | 기대 효과 |
|------|-----------|-----------|-----------|
| **1단계 (즉시)** | 딥링크 연동 | 가맹점별 앱 링크 정보 | 사용자가 EasyToGo에서 식당 발견 → 공공배달앱으로 이동하여 주문 |
| **2단계 (6개월)** | 웹뷰 연동 | 주문 페이지 URL 공유 | 앱 전환 없이 EasyToGo 내에서 공공배달앱 주문 완료 |
| **3단계 (12개월)** | 표준 API 연동 | 2023년 배달 표준 API 포맷 적용 | 메뉴 조회→주문→결제→배달 추적 전 과정을 EasyToGo 내 완결 |

#### 공공배달앱 측 이점

```
현재 공공배달앱의 최대 과제는 "이용자 확보"입니다.
(10여 곳의 지자체 공공배달앱이 낮은 이용률로 폐지됨)

EasyToGo와의 연동은 연간 2,500만 방한 외국인이라는 
완전히 새로운 고객층을 공공배달앱에 유입시킵니다.

외국인 관광객은 배달의민족·쿠팡이츠를 사용하기 어렵습니다
(한국어 전용 + 한국 번호 필수). 공공배달앱이 외국인 전용 
채널을 확보하면, 민간 앱과의 차별화 포인트가 됩니다.
```

> 이 연동 협의를 위해 NHN페이코(배달특급 운영사) 또는
> 농림축산식품부 공공배달 통합포털 담당 부서와의
> 미팅을 요청드립니다.

## 신청서 2: 외국인환자 유치기관 DB

### 기본 정보
- **신청자**: 김제언 (EasyToGo 대표)
- **연락처**: jaon14@gmail.com / 010-2760-6261
- **신청 유형**: 데이터 개방 신청 (신규)

### 개방 요청 데이터

- **대상 기관**: 한국보건산업진흥원 (KHIDI)
- **요청 데이터명**: 외국인환자 유치 등록 의료기관 및 유치업체 정보
- **요청 데이터 항목**:
  1. 의료기관/유치업체 등록번호
  2. 기관명 (한글 + 영문)
  3. 소재지 주소 + 위도·경도
  4. 진료과목 (등록된 외국인 대상 과목)
  5. 대응 가능 언어 (영어/중국어/일본어/러시아어 등)
  6. 연락처 (대표 전화/이메일)
  7. 등록 상태 (유효/만료/취소)
  8. 유치업체의 경우: 연계 의료기관 목록
- **요청 데이터 형식**: REST API (JSON)
- **참고**: KHIDI "외국인환자 유치의료기관 검색" 서비스에서 일부 정보가 웹으로 제공되나, API 형태의 체계적 데이터 접근이 불가

### 활용 목적

```
외국인 환자의 한국 의료기관 접근성 개선을 위한 다국어 의료관광 
플랫폼 개발에 활용하고자 합니다.

현재 외국인 환자가 한국에서 진료받으려면:
(1) 등록된 유치기관/병원을 직접 검색 (한국어)
(2) 중개업체를 별도로 찾아 연락
(3) 보험 서류를 수작업으로 처리
하는 과정을 거쳐야 합니다.

해당 데이터가 API로 개방되면, 외국인 환자가 모국어로 
검증된 의료기관을 검색하고 예약까지 원스톱으로 진행할 수 있어,
「제2차 의료 해외진출 및 외국인환자 유치 지원 종합계획」의
2026년 외국인 환자 50만 명 유치 목표 달성에 기여합니다.
```

### 기대 효과
1. 외국인 환자의 검증된 의료기관 접근성 향상 (무등록 브로커 피해 예방)
2. KHIDI 등록 의료기관의 글로벌 노출 확대
3. 의료관광 산업의 디지털 전환 촉진
4. 외국인환자 유치 50만 명 정책 목표 지원

### 기본 정보

- **연락처**: jaon14@gmail.com / 010-2760-6261
- **신청 유형**: 기존 데이터 항목 추가 요청

### 개방 요청 데이터

- **대상 기관**: 한국관광공사 (KTO)
- **기존 데이터**: TourAPI 4.0 (이미 공개)
- **추가 요청 항목**:
  1. 관광지 내부 시설물 위치 좌표 (건물 내 층별 안내 포함)
  2. 실시간 혼잡도 정보 (방문자 수 기반)
  3. 관광지 주변 소상공인 정보 연계 (반경 500m 내 식당/카페)
  4. 유네스코 유산 등재 관광지의 상세 해설 텍스트 (영/중/일)
  5. 관광지별 대중교통 접근 경로 (최적 루트)
- **요청 데이터 형식**: 기존 TourAPI에 추가 항목으로 포함

### 활용 목적

```
AR(증강현실) 기반 관광 안내 서비스 개발에 활용하고자 합니다.

외국인 관광객이 카메라를 관광지에 비추면 AR 오버레이로 
해당 장소의 역사, 주변 맛집, 교통편을 모국어로 즉시 
안내하는 서비스를 구축 중입니다.

현재 TourAPI는 기본적인 관광지 정보(명칭/주소/소개)를 
제공하지만, AR 서비스에 필요한 정밀 좌표 및 실시간 정보가 
부족합니다. 해당 데이터가 추가되면 KTO의 AX(AI 전환) 
전략과 직결되는 혁신 서비스 사례가 될 수 있습니다.
```

### 기대 효과
1. KTO TourAPI 활용 사례 확대 (AX 전략 성과)
2. 외국인 관광객 체류 만족도 향상 → 재방문율 증가
3. 관광지 주변 소상공인 매출 연계 효과
4. AR 관광 콘텐츠 글로벌 경쟁력 확보

- **요청 데이터 항목**:

```
현금으로 충전해야 하며, 어디서 충전 가능한지, 외국 카드가 
되는지 사전에 알 수 없습니다.

해당 데이터가 개방되면 "가장 가까운 교통카드 충전소" 및 
"외국 카드 사용 가능한 환전소"를 실시간으로 안내할 수 있어,
외국인 관광객의 이동 편의가 크게 개선됩니다.
```
