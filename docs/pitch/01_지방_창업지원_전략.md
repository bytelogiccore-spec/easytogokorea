## ⭐ EasyToGo에 최적인 지방 프로그램 (적합도 순)

### 1. 🏔️ 강원 관광 벤처 지원 사업 — ★★★★★ (최우선)

| 항목 | 내용 |
|------|------|
| **주관** | 강원창조경제혁신센터 |
| **내용** | AX 기반 관광기술 및 혁신 관광 콘텐츠 스타트업 모집 |
| **왜 적합한가** | EasyToGo의 AR 관광 + AI 번역이 **정확히** 이 사업의 타겟 |
| **어필 포인트** | "강원도 스키 리조트·속초·강릉 외국인 관광객을 위한 AR 네비게이션" |
| **지역 특화 피치** | 평창 올림픽 인프라 재활용 + 강원 관광특구 디지털 전환 |

**접근법:**
> "강원도에는 평창 올림픽 이후 세계적 수준의 관광 인프라가 있지만, 외국인 관광객이 독립적으로 이용하기 어렵습니다. EasyToGo는 AR 오버레이로 강원도의 스키 리조트, 해안 관광지, 전통시장을 외국인이 **앱 하나로** 체험할 수 있게 만듭니다."

---

### 2. 🏛️ 안동시 청년 스타트업 관광기업 — ★★★★★ (최우선)

| 항목 | 내용 |
|------|------|
| **주관** | 안동시 + (재)한국정신문화재단 |
| **지원금** | 선정 기업 최대 **3,000만 원** 사업화 자금 |
| **왜 적합한가** | 관광 + 스타트업 + 지역 활성화의 교차점 |
| **어필 포인트** | "하회마을·안동찜닭골목 등 전통문화 관광지의 외국인 접근성" |
| **지역 특화 피치** | 유네스코 세계유산(하회마을, 도산서원) AR 해설 투어 |

**접근법:**
> "안동은 한국 전통문화의 핵심이지만, 외국인 여행자가 가이드 없이 방문하기에는 정보 장벽이 높습니다. EasyToGo의 AR 해설 기능으로 하회마을의 역사를 걸으면서 영어·중국어·일본어로 체험할 수 있습니다. 전통시장 메뉴판도 실시간 번역됩니다."

### 3. 🌾 광주·전남 로컬창업기업 육성 (LIPS) — ★★★★☆

| 항목 | 내용 |
|------|------|
| **주관** | 광주창조경제혁신센터 (호남권역 운영사) |
| **내용** | 광주·전남 로컬 창업기업 발굴·육성 프로그램 |
| **왜 적합한가** | "로컬 창업" = 지역 소상공인 지원과 EasyToGo의 SaaS 모델 일치 |
| **어필 포인트** | "전남 소상공인에게 수수료 0% 디지털 매장을 제공" |
| **지역 특화 피치** | 광주 5·18 역사문화 관광 + 전남 남도 음식 관광 |

**접근법:**
> "남도 음식은 한국 관광의 숨겨진 보석이지만, 외국인이 순천·여수·목포의 맛집을 찾으려면 네이버 지도 한국어 검색이 필수입니다. EasyToGo는 소상공인에게 자체 호스팅 다국어 매장을 제공하고, 관광객에게는 AR로 메뉴판을 자동 번역합니다."

## 🎯 지역별 EasyToGo 피치 커스터마이징 가이드

모든 지역 피치에 공통으로 들어가야 할 내용:

```
[공통 프레임워크]
1. 문제: "한국은 세계 최고 인프라인데, 외국인은 이용할 수 없다"
2. 솔루션: EasyToGo = Future City OS (AR + SaaS + Medical)
3. 기술: Tauri V2 + Svelte + Rust WASM + On-device AI
4. 수익: B2B SaaS (관광객은 무료)
```

각 지역에는 **로컬 어필 포인트**를 교체:

| 지역 | 로컬 키워드 | 피치 포인트 |
|------|-------------|-------------|
| **강원** | 스키, 평창, 속초, 강릉 | "동계 스포츠 관광 + 해안 도시 디지털화" |
| **안동/경북** | 하회마을, 경주, 한방 | "유네스코 유산 AR 해설 + 한방 의료관광" |
| **광주/전남** | 남도 음식, 여수, 순천 | "미식 관광의 언어 장벽 해소 + 소상공인 SaaS" |
| **충청** | 천안/아산 병원, KTX | "수도권 접근성 + 의료클러스터 디지털 전환" |
| **제주** | 올레길, 독립 관광 | "렌터카 관광객 AR 내비 + 제주 QR 버스" |
