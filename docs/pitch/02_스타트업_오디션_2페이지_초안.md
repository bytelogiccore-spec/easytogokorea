### 1. 문제: 한국은 최고의 인프라, 최악의 접근성

한국은 세계 최고 수준의 교통·의료·IT 인프라를 보유하고 있지만,
**외국인 관광객은 그것을 이용할 수 없습니다.**

| 영역               | 외국인이 겪는 현실                                              |
| ------------------ | --------------------------------------------------------------- |
| 🚇**교통**   | 교통카드 현금 충전만 가능. QR 결제는 제주만 시범 운영           |
| 💳**결제**   | 외국 카드 결제 실패율 높음. 환전은 은행 영업시간에만 가능       |
| 🗣️**언어** | 메뉴판·의료 서류 90% 이상 한국어 전용                          |
| 🏥**의료**   | K-메디컬은 세계적 수준이지만, 보험·중개업체 장벽으로 접근 불가 |
| 📱**정보**   | 필수 정보가 10개 이상 앱에 파편화                               |

**2025년 방한 외국인 2,500만 명, 관광 수입 30조 원 목표 (KTO 2026).**
하지만 외국인 "재방문율"은 인프라 접근성 문제로 정체 중.

---

### 2. 솔루션: EasyToGo = 도시를 읽는 OS

EasyToGo는 **하나의 앱**으로 한국의 모든 인프라를 외국인에게 연결합니다.

#### 🔮 AR 오버레이 — 도시가 말을 걸다

- 카메라를 대면 간판·메뉴판·지하철 안내가 **자동 번역**
- 유네스코 유적지에서 AR 해설 투어 (경주, 안동, 수원)
- AR 그래피티 SNS — 관광지에 디지털 아트 남기기

#### 🏪 소상공인 SaaS — 수수료 0%의 디지털 독립

- Tauri V2 기반 **자체 호스팅** 솔루션 (배달앱 수수료에서 해방)
- 다국어 디지털 메뉴·주문·결제 시스템
- 소상공인이 자기 데이터를 소유 (Shopify for Tourism)

#### 🏥 의료 자동화 — K-메디컬의 라스트마일

- 병원 발견 → 다국어 접수 → AI 통역 → 전자 보험 청구
- **No-Storage 아키텍처**: 의료 데이터는 기기 내 AI로만 처리, 서버 저장 0
- 의료 에이전시 B2B2C 마켓플레이스 (에이전시·병원·환자 3자 매칭)

### 3. 기술 차별성

| 기술                   | 적용                   | 왜 중요한가                                           |
| ---------------------- | ---------------------- | ----------------------------------------------------- |
| **Svelte + Tauri V2** | 크로스플랫폼 앱        | iOS/Android 동시 배포, 1인 개발 효율 극대화           |
| **Rust**         | 백엔드 + AI 파이프라인 | 메모리 안전성 + C++ 급 속도, 의료 데이터 처리에 적합  |
| **Tauri V2**     | 소상공인 자체호스팅    | 웹 기반 경량 데스크탑 앱, 소상공인 PC에서 직접 실행   |
| **On-device AI** | OCR + 번역 + 의료 처리 | 개인정보를 서버에 보내지 않는 "No-Storage" 프라이버시 |
| **GraphQL**      | API 통합               | KTO·기상청·교통 등 10+ 공공 API를 하나로 통합       |
